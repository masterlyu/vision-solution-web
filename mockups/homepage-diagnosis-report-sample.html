Executive Summary

지금 당장 고객을 잃고 있는 이유 3가지

📵
전화 연결이 안 됩니다
모바일에서 전화번호를 탭해도 전화가 걸리지 않습니다. 예약하려던 고객이 그냥 닫습니다.
👁️
첫 화면이 텅 비어 있습니다
방문자가 3초 안에 "여기가 어떤 식당인지"를 알 수 없습니다. 차별점이 보이지 않습니다.
🐌
6.8초가 걸립니다
구글 권장 기준(3초)의 2배 이상입니다. 모바일 방문자 53%가 로딩 중 이탈합니다.
핵심 개선 항목

AS-IS → TO-BE

현재 상태와 개선 후 기대 효과를 항목별로 정리했습니다.

디자인·UX 첫 화면 메시지 🔴 즉시 개선 필요
AS-IS (현재)

"오신 것을 환영합니다" 텍스트와 대형 배너 이미지만 있음. 어떤 식당인지, 무엇이 특별한지, 어떻게 예약하는지 스크롤 없이 알 수 없음.

→
TO-BE (개선 후)

헤드라인: "30년 전통 한정식, 강남 유일 코스 전문" + 서브텍스트로 특징 2줄 + "지금 예약하기" 버튼이 첫 화면 안에 모두 위치.

💡 예상 효과: 첫 화면 이탈률 25~35% 감소, 예약 문의 클릭률 2배 이상 향상
기술 모바일 전화 연결 🔴 즉시 개선 필요
AS-IS (현재)

전화번호가 일반 텍스트로 표시됨. 모바일에서 탭해도 전화 앱이 열리지 않음. 복사 후 직접 전화해야 함.

02-1234-5678
→
TO-BE (개선 후)

tel: 링크로 변경. 모바일에서 탭하면 즉시 전화 앱 실행. 상단 고정 바에 전화 버튼 상시 노출.

<a href="tel:0212345678">전화하기</a>
💡 예상 효과: 모바일 전화 문의 40~60% 즉시 증가 (수정 소요 시간: 10분)
기술 페이지 로딩 속도 🟡 개선 권장
AS-IS (현재)
6.8초 모바일 기준

최적화 안 된 이미지 8장 (합계 14MB), 사용 안 하는 플러그인 11개, CSS/JS 미압축

→
TO-BE (개선 후)
1.8초 목표

이미지 WebP 변환·압축, 불필요 플러그인 제거, CSS/JS 번들링, 브라우저 캐시 설정

💡 예상 효과: 모바일 이탈률 30% 감소, 구글 검색 순위 개선 (속도는 SEO 랭킹 요소)
영업 의도 CTA 설계 (예약 유도) 🔴 즉시 개선 필요
AS-IS (현재)

예약 버튼이 하단 메뉴에만 있음. 스크롤을 4~5번 내려야 도달. 버튼 색상이 배경과 구분이 안 됨. 클릭 유도 문구 없음.

→
TO-BE (개선 후)

헤로 영역 + 메뉴 소개 하단 + 페이지 하단 3곳 배치. "지금 예약하면 웨이팅 없이 입장" 문구로 긴박감 부여. 고대비 버튼으로 시인성 확보.

💡 예상 효과: 예약 클릭률 2~3배 향상, 평균 체류 시간 증가
영업 의도 신뢰 요소 🟡 개선 권장
AS-IS (현재)

고객 후기 없음. 방문 인증 사진 없음. 언론 보도·수상 이력 없음. "30년 전통"이라고만 텍스트로 적혀 있어 신뢰도가 낮음.

→
TO-BE (개선 후)

네이버 리뷰 별점 위젯 연동, 실제 고객 후기 3~5개 카드, 방문객 포토월, "OO 맛집" 언론 노출 배지 추가.

💡 예상 효과: 초방문자 신뢰도 향상, 예약 전환율 20~30% 증가
