GitHub에서 별 35만 개를 넘긴 프로젝트가 있습니다.

서울 마포구에서 프리랜서 디자인 일을 하는 A씨(38세)는 새벽 2시에 화면을 멍하니 바라봤습니다. "이 프로젝트, 5개월 만에 별 35만 개?" 댓글창은 온통 "나도 됐어요!", "진짜 무료예요?" 였습니다.

그 프로젝트 이름은 **OpenClaw**입니다.

ChatGPT를 쓰려면 사이트 들어가서 로그인해야 하죠. 매번 탭 열기도 귀찮습니다. OpenClaw는 그 불편함을 없앴어요. 쓰던 메신저 안에서 그냥 말 걸면 AI가 대답합니다.

## 별 35만 개, 이게 얼마나 대단한 건가

유명 오픈소스 프로젝트들이 GitHub 별 10만 개를 모으는 데 보통 2~3년 걸립니다.

OpenClaw는 달랐습니다. 2025년 11월에 처음 공개됐어요. 오스트리아 개발자 Peter Steinberger가 만든 작은 프로젝트였는데, 입소문이 엄청났습니다. 2026년 4월 기준으로 별이 35만 개를 넘었어요. 기여자(함께 개발에 참여한 사람)도 전 세계에서 1,600명 넘게 모였습니다.

GitHub 별이란 간단히 말하면 "좋아요"입니다. IT 개발자들이 "이 프로그램 쓸 만하다"고 누르는 거예요.

35만 명이 누른 겁니다. 반년 만에.

## OpenClaw가 뭔데 이렇게 인기인가

쉽게 설명하면 이렇습니다.

ChatGPT 같은 AI를 **텔레그램·카카오워크·디스코드** 안에서 쓸 수 있게 해주는 연결 프로그램이에요. 새 앱 설치? 필요 없어요. 이미 쓰고 있는 메신저에서 AI한테 말 걸면 됩니다.

누구나 무료로 쓸 수 있는 오픈소스(누구나 무료로 쓸 수 있는 공개 프로그램)입니다. 텔레그램, WhatsApp, Discord, Slack, Signal, iMessage 등 20개가 넘는 메신저와 연결됩니다. 한국에서 많이 쓰는 텔레그램으로 시작하면 딱입니다.

그리고 AI가 이런 일을 대신 해줍니다.

| 하고 싶은 것 | OpenClaw에 시키는 말 |
|------|------|
| 뉴스 요약 | "오늘 주요 뉴스 3가지 알려줘" |
| 문서 정리 | "이 계약서 핵심만 정리해줘" |
| 번역 | "이 영어 이메일 한국어로 바꿔줘" |
| 자료 검색 | "이 사이트에서 가격 정보 뽑아줘" |

비용은 AI API 사용료만 냅니다. 처음 가입하면 무료 크레딧을 줘서 첫 달은 돈 없이도 됩니다.

## 텔레그램에서 10분 만에 시작하는 법

자, 실제로 해봅시다. 준비물은 딱 두 가지입니다.

**필요한 것:** 텔레그램 앱, 무료 AI API 키

혹시 이런 경험 있으세요? 유튜브 보며 따라 했는데 중간에 막혀서 포기한 경험. 여기선 최대한 쉽게 설명할게요.

**① 텔레그램 봇 만들기**

텔레그램을 열고 검색창에 `@BotFather`를 찾습니다. 대화창에서 `/newbot`이라고 보내세요. 봇 이름을 물어보면 원하는 걸 입력합니다. 그러면 **API 토큰**이라는 긴 숫자 코드를 줍니다. 메모해두세요.

**② OpenClaw 설정 파일 받기**

[openclaw.ai](https://openclaw.ai) 사이트에서 `config.yml` 파일을 내려받습니다. 메모장으로 열어서 아까 받은 텔레그램 토큰을 붙여넣으면 됩니다.

**③ 무료 AI 연결하기**

Claude AI의 API 키는 anthropic.com에서 무료로 받습니다. 신규 가입 시 5달러 크레딧을 줘요. 이 키도 config.yml에 넣으세요.

**④ 실행**

```
docker run -d openclaw/openclaw
```

이 한 줄이면 끝납니다. 이제 텔레그램에서 내 봇한테 "안녕"하면 AI가 대답해요.

처음 해도 15분이면 충분합니다.

## 아무도 안 알려준 꿀팁 2가지

**꿀팁 1 — 무료 크레딧 다 쓰면?**

Claude API 무료 크레딧이 바닥나도 걱정 없습니다. OpenClaw는 **Groq**이라는 서비스와도 연결됩니다. Groq는 응답 속도가 매우 빠른 무료 AI 서비스예요. 월 무료 한도가 상당히 넉넉합니다.

config.yml 파일에서 `provider: groq`로 한 줄만 바꾸면 됩니다. 완전히 무료로 계속 쓸 수 있어요.

**꿀팁 2 — 한국어 성능 올리는 첫 마디**

AI 봇을 처음 실행한 뒤, 이 문장을 먼저 보내세요.

> "지금부터 모든 답변은 한국어로 해줘. 어려운 전문용어는 쉬운 말로 풀어서 설명해줘."

이게 AI에게 미리 주는 지시(프롬프트)가 됩니다. 이후 모든 대화가 쉬운 한국어로 돌아와요. 영어로 설정된 기본값을 내 입맛대로 바꾸는 거예요.

## 이런 실수 하지 마세요

OpenClaw 처음 설정할 때 자주 막히는 지점이 있습니다.

**실수 1 — API 키 앞뒤 공백 넣기**

복사해서 붙여넣을 때 앞뒤에 공백이 끼면 연결이 안 됩니다. 키 전후로 스페이스가 없는지 확인하세요.

**실수 2 — Docker 없이 실행하려는 것**

OpenClaw는 Docker라는 프로그램이 필요합니다. Docker가 뭔지 모르셔도 됩니다. [docker.com](https://www.docker.com)에서 그냥 내려받아서 설치하면 돼요. 설치 후 재부팅 한 번이면 준비 완료입니다.

**실수 3 — 봇 대화창 찾지 못하기**

BotFather에서 봇을 만든 뒤, 내 봇 이름으로 직접 검색해야 합니다. 대화창을 열고 `/start`를 보내면 시작됩니다. 안 보이면 텔레그램 검색창에서 봇 아이디로 찾아보세요.

## 마무리 — 한 번 만들면 평생 쓴다

그런데 딱 한 번만 하면 됩니다. 그다음부터는 텔레그램 열고 말 거는 것뿐이에요. AI 비서가 24시간 기다리고 있는 거니까요.
