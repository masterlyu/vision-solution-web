## 우리 회사엔 어떻게 씁니까

슬랙을 쓰는 중소기업이라면 지금 당장 이런 상황에서 활용할 수 있습니다.

**케이스 1 — 회의 스레드 요약**
팀장이 퇴근 전 "@클로드 오늘 #마케팅채널 대화 정리해줘" 하고 태그합니다.
클로드가 스레드를 읽고 결정 사항·담당자·기한을 목록으로 뽑습니다.

반대도 됩니다. 한국어로 요청 내용을 쓰면 비즈니스 영어 이메일로 바꿔줍니다.

**케이스 3 — 직원 반복 질문 자동 처리**

"연차는 어떻게 써요?", "야근 수당 기준이 뭐예요?" 같은 질문에
클로드가 먼저 답합니다. 담당자는 예외 케이스만 처리하면 됩니다.

**케이스 4 — 보고서 초안 작성**
숫자와 방향성만 채팅으로 던지면 클로드가 보고서 틀을 잡아줍니다.

## 지금 따라하기 — 3분 설치

개발 지식 불필요합니다. 아래 단계를 순서대로 따라가시면 됩니다.

**① 슬랙 앱 디렉터리 열기**
슬랙 좌측 메뉴 하단 "앱 추가" 클릭, 또는 [slack.com/apps](https://slack.com/apps) 접속.
검색창에 "Claude" 입력.

**② Claude for Slack 선택 후 설치**
"Claude for Slack" (Anthropic 공식) 클릭 → "Add to Slack" 버튼.
워크스페이스 관리자 권한이 필요합니다. 본인이 관리자가 아니라면 관리자에게 요청하세요.

**③ Claude.ai 계정 연결**
계정이 없다면 [claude.ai](https://claude.ai)에서 무료 가입 후 진행합니다. 이메일 하나면 됩니다.

**④ 채널에 클로드 초대**
원하는 채널로 이동 → 메시지 입력창에 아래를 입력하고 엔터.

```
/invite @claude
```

**⑤ 첫 사용**
```
@claude 이 스레드 핵심만 3줄로 요약해줘
```
이렇게 보내면 클로드가 답합니다. 끝입니다.

### 바로 복붙해서 쓰는 프롬프트 4가지

- **회의 요약**: `@claude 위 대화 요약해줘. 결정 사항, 담당자, 기한을 구분해서`
- **이메일 번역**: `@claude 이 영어 이메일 한국어로 번역해줘`
- **보고서 초안**: `@claude 다음 내용으로 팀장 보고용 1페이지 요약 작성해줘: [내용 붙여넣기]`
- **사내 Q&A**: `@claude 우리 회사 연차 정책이 뭐야? [정책 내용 붙여넣기]`
