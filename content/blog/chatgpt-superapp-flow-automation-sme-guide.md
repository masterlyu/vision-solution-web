## ChatGPT가 달라진 것, 정확히 뭔가

"슈퍼앱"이란 카카오처럼 **하나의 앱 안에 여러 기능이 통합된 앱**을 말합니다. 카카오는 메시지·결제·쇼핑을 한 곳에 담았고, ChatGPT는 이제 대화·자동화·앱 연동을 한 번에 처리하게 됐습니다.

이번 개편의 핵심 세 가지를 짚겠습니다.

**ChatGPT Flow** — 드래그&드롭으로 만드는 워크플로우 빌더입니다. 코딩 없이 "이메일이 오면 → 내용을 분류하고 → 슬랙에 알림"처럼 업무 흐름을 그림으로 설계하면 ChatGPT가 알아서 실행합니다. Zapier의 "Zap"과 개념이 같지만 ChatGPT 안에 바로 내장됐습니다.

**OpenAI Operator** — ChatGPT가 직접 브라우저를 조작합니다. 예약·검색·주문처럼 반복되는 웹 작업을 자동으로 처리합니다. 2026년 현재 한국을 포함한 40개국 이상에서 사용 가능합니다.

**AgentKit** — 코딩 없이 AI 에이전트를 설계하는 도구입니다. OpenAI에 따르면 처음 셋업하는 데 2시간이면 충분합니다.

세 기능 모두 Canva, Figma, Slack, Notion, Booking.com 등 **90개 이상의 외부 서비스와 연동**됩니다.

## 중소기업이 지금 없앨 수 있는 반복 업무 3가지

![center](/mascot/md/emotion/cat_thinking.webp)

중소기업 현장에서 시간을 가장 많이 잡아먹는 반복 업무 세 가지를 골랐습니다.

### ① 고객 이메일 자동 분류

**지금 상황**: 하루에 견적 문의, 클레임, FAQ가 섞인 이메일 수십 통이 받은편지함에 뒤섞입니다. 담당자가 하나씩 열어보고 분류하는 데 하루 1~2시간이 날아갑니다.

**자동화 후**: 이메일이 수신되는 순간 ChatGPT Flow가 내용을 읽고 분기합니다. "견적 문의"는 영업팀 슬랙 채널로, "클레임"은 CRM 긴급 티켓으로, "FAQ"는 자동 답장으로 처리됩니다. 담당자는 분류가 끝난 항목만 처리하면 됩니다.

### ② 견적서 자동 생성

**지금 상황**: 고객이 상품·수량을 보내면 담당자가 엑셀 단가표를 열고, 금액을 계산하고, PDF로 변환해 이메일로 발송합니다. 건당 20~30분 짜리 작업이 하루에 수십 번 반복됩니다.

**자동화 후**: 고객 이메일이나 폼 응답이 들어오면 → ChatGPT가 단가표를 참조해 견적서 초안을 생성하고 → PDF 변환 후 담당자에게 검토용으로 전달합니다. 담당자는 확인 후 클릭 한 번으로 발송. 건당 작업 시간이 20분에서 3분으로 줄어듭니다.

### ③ SNS 예약 발행

**지금 상황**: 마케팅 담당자가 인스타그램용, 페이스북용, 블로그용 글을 별도로 작성하고 각각 수동으로 업로드합니다. 같은 내용을 형식만 바꿔 3번 반복하는 셈이죠.

**자동화 후**: 초안 하나를 ChatGPT에 넣으면 → 각 플랫폼 톤에 맞게 자동 변환 → Notion·Buffer와 연동해 예약 발행. 3개 채널 발행 작업이 초안 한 번으로 끝납니다.

## 지금 바로 따라하는 실습 — 이메일 자동 분류 만들기

![center](/mascot/md/emotion/cat_surprised.webp)

ChatGPT Plus 계정이 있으면 지금 바로 시작할 수 있습니다.

**1단계 — ChatGPT에서 Flow 열기**

chatgpt.com 접속 → 왼쪽 사이드바 또는 새 대화 화면에서 **"Flow"** 탭 클릭 → "새 워크플로우" 선택.

**2단계 — 아래 프롬프트 붙여넣기**

```
고객 이메일을 자동으로 분류하는 워크플로우를 만들어 줘.
규칙:
- 견적 관련 → 제목에 [견적] 태그 + 영업팀 슬랙 채널에 알림
- 불만·클레임 → [긴급] 태그 + 담당자 즉시 이메일 알림
- 일반 문의 → 표준 자동 답장 발송
연동 앱: Gmail, Slack
```

**3단계 — 앱 연동 승인 후 테스트**

Flow가 Gmail과 Slack 연결 권한을 요청합니다. 승인 후 테스트 이메일을 보내 워크플로우가 정상 실행되는지 확인합니다. 여기까지 보통 30~60분이면 됩니다.
