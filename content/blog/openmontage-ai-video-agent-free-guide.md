## "외주 맡겼다가 150만 원 날렸어요" — 홍보영상의 진짜 문제

경기도에서 반찬 가게를 운영하는 B씨는 작년에 홍보영상 한 편을 외주로 맡겼습니다.

기획부터 촬영, 편집까지 포함하니 견적이 150만 원이었습니다. 3주를 기다렸고, 결과물은 2분짜리 영상 하나. 유튜브에 올렸지만 조회수는 100을 넘지 않았습니다. "다음에는 못 하겠다"는 생각이 들었다고 합니다.

혹시 비슷한 경험 있으신가요? 필요한 건 알지만, 비용과 시간 때문에 영상 마케팅을 포기한 적 말입니다.

2026년 6월, 이 상황이 달라질 수 있는 도구가 나왔습니다. 이름은 **OpenMontage** — 세계 최초 오픈소스 에이전틱 영상 제작 시스템입니다. 기획, 스크립트 작성, 영상 소재 생성, 편집, 렌더링까지 AI 에이전트가 처음부터 끝까지 자동으로 처리합니다. 소프트웨어 비용은 0원입니다.

## "에이전틱"이란 뭔가요? — AI가 알아서 하는 영상 제작

OpenMontage를 이해하려면 먼저 '에이전틱(Agentic)'이라는 개념이 필요합니다.

쉽게 설명하면 이렇습니다. 기존 AI 도구는 "이 문장을 번역해줘"처럼 한 가지 작업만 시킬 수 있었습니다. 에이전틱 AI는 다릅니다. "우리 가게 신메뉴 홍보영상 만들어줘"라고 하면, AI가 스스로 계획을 세우고, 필요한 소재를 만들고, 편집하고, 최종 파일까지 내놓습니다. 사람이 중간에 일일이 지시하지 않아도 됩니다.

![center](/mascot/md/emotion/cat_happy.webp)

OpenMontage는 이 방식으로 영상 제작의 전 과정을 자동화합니다. 지원하는 영상 유형만 12가지입니다. 제품 설명 영상, 발표자가 나오는 토킹헤드 영상, 화면 데모 영상, 애니메이션, 팟캐스트 형식, 다큐멘터리 스타일, 다국어 번역 영상까지 다양합니다.

2026년 3월 GitHub에 공개된 뒤 가파르게 성장해, 6월에는 GitHub 트렌딩 1위에 올랐습니다. 현재 별(스타)은 2만 6천 개를 넘었습니다. GitHub 스타는 개발자들이 "이거 쓸만하다"고 표시하는 방식인데, 이 수치는 그만큼 주목받고 있다는 뜻입니다.

## 설치와 요구 사항 — 솔직하게 따져봅니다

기대가 크면 현실 확인이 먼저입니다. OpenMontage가 작동하려면 몇 가지가 갖춰져야 합니다.

**필수 환경**

| 항목 | 요구 사항 |
|------|---------|
| Python | 3.10 이상 |
| FFmpeg | 설치 필요 (무료) |
| Node.js | 18 이상 |
| AI 코딩 어시스턴트 | Claude Code, Cursor, Copilot 등 1개 |

**AI 코딩 어시스턴트**란 개발자들이 쓰는 AI 도구입니다. Cursor(무료 플랜 있음)나 GitHub Copilot(월 약 14달러)처럼 코드 편집기에 붙어 AI와 대화하며 작업하는 환경입니다. OpenMontage는 이 환경에서 자연어 프롬프트를 받아 영상을 제작합니다.

![center](/mascot/md/emotion/cat_thinking.webp)

**영상 품질과 GPU**

영상을 직접 생성(WAN 2.1, Hunyuan, CogVideo 등 로컬 모델)하려면 고성능 GPU가 필요합니다. 일반 사무용 PC로는 어렵습니다. GPU가 없다면 외부 AI 서비스(API)를 연결해 쓸 수도 있는데, 이 경우 영상 생성에 별도 API 비용이 발생합니다.

나레이션은 **Piper TTS**라는 무료 오프라인 엔진을 기본으로 사용합니다. 인터넷 없이 텍스트를 음성으로 바꿀 수 있어 추가 비용이 들지 않습니다.

**솔직한 장단점**

| 구분 | 내용 |
|------|------|
| 장점 | 소프트웨어 자체 무료, 영상 제작 전 과정 자동화, 12가지 영상 유형 지원 |
| 단점 | 고사양 GPU 없으면 로컬 영상 생성 어려움, 설치 과정이 비개발자에게 다소 복잡 |
| 대안 | 로컬 GPU 없으면 Runway ML(클라우드 유료), Pika(클라우드 유료) 비교 검토 필요 |

GPU가 없는 환경이라면 클라우드 기반 유료 서비스와 비교해 선택하는 게 현실적입니다. 다만 OpenMontage의 전체 자동화 파이프라인(기획→편집→렌더링)은 현재 유사 경쟁 서비스에 없는 기능입니다.

그럼 실제로 어떤 업무에 쓸 수 있을까요?

## 우리 가게 홍보영상, 실제로 어떻게 쓰나

중소기업·소상공인 관점에서 실제 활용 시나리오를 정리했습니다.

**시나리오 ① 식음료 가게 — 신메뉴 소개 영상**

제품 사진 몇 장과 함께 "우리 가게 봄 신메뉴 딸기 파르페 소개 영상을 만들어줘. 30초 분량, 밝은 배경음악 포함, 자막 있게"라고 입력합니다. AI 에이전트가 스크립트를 쓰고, 사진을 이용해 영상을 편집하고, 자막과 음악을 붙입니다. SNS 업로드용 영상이 나옵니다.

**시나리오 ② 제조업체 — 제품 데모 영상**

화면 녹화나 제품 사진을 넣고 "우리 설비 운영 방법 3분 소개 영상 만들어줘"라고 하면, 화면 데모 파이프라인이 작동해 설명 자막과 나레이션이 붙은 데모 영상이 나옵니다. 납품처나 바이어에게 보내는 소개 자료로 쓸 수 있습니다.

**시나리오 ③ 서비스업 — 다국어 고객 안내 영상**

"우리 식당 이용 방법 안내 영상을 한국어·영어·중국어 버전으로 만들어줘"라고 하면 로컬라이제이션 파이프라인이 자동으로 3개 언어 버전을 제작합니다. 외국인 고객이 많은 사업장에서 바로 쓸 수 있습니다.

![OpenMontage 자동화 파이프라인](/images/blog/openmontage-ai-video-agent-free-guide-fig2.svg)

회사 데이터(제품 사진, 브랜드 자료, 기존 문서)를 OpenMontage 폴더에 넣어두고 자연어로 요청하면 됩니다. 반복 영상 제작이 필요한 업무 — 매달 신상품 소개, 분기별 회사 소식, 시즌별 이벤트 안내 — 를 자동화할 수 있습니다.

## 지금 바로 따라할 수 있는 실습 3단계

아래 과정은 Python 3.10 이상, Node.js 18 이상, FFmpeg가 이미 설치된 환경 기준입니다. Cursor(무료 버전 가능) 또는 Claude Code 중 하나를 준비해 두세요.

**1단계 — 저장소 내려받기**

터미널(윈도우는 명령 프롬프트 또는 PowerShell)을 열고 아래 명령어를 입력합니다.

```bash
git clone https://github.com/calesthio/OpenMontage.git
cd OpenMontage
make setup
```

설치 완료 메시지가 뜨면 준비 끝입니다.

**2단계 — AI 어시스턴트에서 폴더 열기**

Cursor를 사용한다면 "File → Open Folder"로 방금 내려받은 OpenMontage 폴더를 엽니다. Claude Code라면 터미널에서 해당 폴더 위치에서 `claude` 명령어를 실행합니다.

**3단계 — 자연어로 영상 요청하기**

AI 어시스턴트 채팅창에 원하는 영상을 설명합니다. 예시:

```
우리 카페 신메뉴 '망고 라씨' 소개 영상을 만들어줘.
30초 분량, 밝고 경쾌한 배경음악, 한국어 자막 포함.
이미지 파일은 images/ 폴더에 있어.
```

AI 에이전트가 스크립트를 짜고, 이미지로 영상을 편집하고, 나레이션과 자막을 붙인 결과 파일을 만들어 줍니다.

## (주)비젼솔루션의 관점 — 영상도 민주화되고 있다

영상 제작이 50~200만 원짜리 외주 작업에서 자연어 한 줄짜리 작업으로 바뀌고 있습니다. 이건 단순한 비용 절감이 아닙니다.

지금까지 홍보 영상은 "만들 예산이 있는 회사만 쓸 수 있는 도구"였습니다. OpenMontage 같은 오픈소스가 등장하면서, 영상을 자주 만들어야 하는 소상공인 — 신메뉴 소개, 시즌 이벤트, 고객 안내 — 도 비용 걱정 없이 시작할 수 있게 됐습니다. 기술이 소수의 전유물에서 벗어나 도구가 되는 순간, 산업의 경쟁 구도가 달라집니다. 영상 마케팅도 지금 그 지점을 지나고 있습니다.

다만 도구가 무료라고 해서 결과물이 바로 쓸 수 있는 수준이 되지는 않습니다. 프롬프트를 얼마나 잘 쓰느냐, 브랜드 자산(로고·색상·사진)을 얼마나 잘 준비해두느냐에 따라 품질 차이가 큽니다. 도구를 쓰는 것과 잘 쓰는 것 사이의 간격 — 그 부분에서 전문가의 도움이 여전히 필요합니다.

## 자주 묻는 질문

**Q. GPU 없는 일반 PC에서 사용할 수 있나요?**

로컬 영상 생성 모델(WAN 2.1, Hunyuan 등)은 고성능 GPU가 필요합니다. GPU가 없다면 외부 AI API를 연결해 쓸 수 있습니다. 이 경우 영상 생성 비용(API 이용료)이 별도 발생합니다.

**Q. OpenMontage로 만든 영상을 상업적으로 써도 되나요?**

OpenMontage 자체는 오픈소스라 무료로 상업 이용 가능합니다. 다만 영상에 사용된 이미지나 음악의 저작권은 별도로 확인해야 합니다. AI 생성 이미지는 대부분 상업 이용이 가능하지만, 외부 소스를 넣었다면 해당 라이선스를 따릅니다.

**Q. 한국어 나레이션이 자연스럽게 나오나요?**

기본 제공 Piper TTS의 한국어 음성은 실용 수준입니다. 더 자연스러운 목소리가 필요하다면 외부 TTS API(ElevenLabs 등)를 연결할 수 있습니다.
