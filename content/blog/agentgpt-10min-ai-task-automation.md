## AI 에이전트, 챗GPT랑 뭐가 다른가요?

![center](/mascot/md/emotion/cat_happy.webp)

챗GPT는 질문하면 답을 주는 도구입니다. 아주 똑똑한 조수인데, 한 가지 약점이 있어요. 시키는 것만 합니다. 다음 단계로 스스로 넘어가지 않아요.

반면 **AI 에이전트(Agent)**는 목표를 주면 스스로 계획을 세우고, 필요한 작업을 순서대로 실행합니다. 중간에 막히면 방법을 바꿔가며 목표를 향해 계속 진행하죠.

비유로 설명하면 이렇습니다. 챗GPT가 "이 이메일 어떻게 쓸까요?"라고 물으면 답 주는 조수라면, AI 에이전트는 "다음 주 고객 미팅 준비해"라고 하면 — 일정 확인 → 관련 자료 수집 → 발표 초안 작성 → 이메일 초안까지 알아서 처리하는 팀원입니다.

**AgentGPT**는 이런 AI 에이전트를 누구나 브라우저에서 바로 만들 수 있게 해주는 오픈소스 도구입니다. Reworkd가 개발해 GitHub에 공개했고, 웹 버전(agentgpt.reworkd.ai)은 설치 없이 즉시 사용 가능합니다.

## 우리 회사엔 어떻게 쓸 수 있을까요?

실제로 중소기업 업무에 적용해보면 이런 시나리오가 가능합니다.

**시나리오 1 — 경쟁사 동향 수집 (마케팅팀)**  
AgentGPT에 목표를 입력합니다: "국내 경쟁사 5곳의 최근 신제품 출시 현황을 정리해줘." 에이전트가 각 회사 웹사이트와 보도자료를 돌아다니며 정보를 모아 요약합니다. 보통 30분 걸리던 자료 수집이 5분 이내로 끝납니다.

**시나리오 2 — 주간 보고서 초안 (경영지원팀)**  
판매 실적 데이터를 붙여넣고 "분석해서 경영진 보고용 요약본 만들어줘"라고 하면 분석 → 요약 → 시사점 도출까지 처리합니다. 물론 최종 검토와 수정은 담당자가 해야 합니다.

**시나리오 3 — 고객 응대 FAQ 초안 (CS팀)**  
"우리 제품의 자주 묻는 질문 15개와 답변 초안을 만들어줘"라고 입력하면 FAQ 문서 초안을 뽑아줍니다. 제품 정보를 함께 붙여넣으면 더 정확한 결과가 나옵니다.

이 모든 결과물은 사람이 검토하고 수정해야 합니다. AI가 80% 수준의 초안을 만들어주면, 나머지 20%는 담당자의 손길로 완성하는 방식이죠. "다 맡긴다"가 아니라 "초안은 AI가, 판단은 사람이" 하는 분업입니다.

![center](/mascot/md/emotion/cat_surprised.webp)

**(주)비젼솔루션 관점** — AI 에이전트 도입에서 가장 흔한 실패는 도구 선택이 아닌 업무 선택의 문제입니다. "가장 반복적이고, 규칙이 명확하고, 실수해도 재작업이 쉬운 업무"부터 시작해야 합니다. 반대로 고객과의 협상 내용이나 법적 판단이 필요한 업무는 아직 AI 에이전트에게 맡기기 이릅니다. 도구의 가능성을 과소평가하지도, 과대평가하지도 않는 균형 감각 — 그게 실제로 AI를 쓰는 회사와 구경만 하는 회사의 차이입니다.

## 지금 바로 10분 실습해보기

프로그램 설치가 필요 없습니다. 순서대로 따라해 보세요.

**1단계 — AgentGPT 접속**  
브라우저에서 [agentgpt.reworkd.ai](https://agentgpt.reworkd.ai)를 엽니다. 회원가입 없이도 기본 체험이 가능합니다.

**2단계 — 에이전트 이름 정하기**  
"Agent Name"란에 역할 이름을 입력합니다. 예: `경쟁사 조사 비서`

**3단계 — 목표 입력**  
"Goal"란에 원하는 작업을 구체적으로 입력합니다. 처음엔 간단한 작업부터 시작해 보세요.

```
국내 중소기업 대상 회계 소프트웨어 3가지를 찾아서
각각의 가격, 주요 기능, 장단점을 비교해줘.
결과는 표 형식으로 정리해줘.
```

**4단계 — 실행 확인**  
"Deploy Agent" 버튼을 클릭합니다. 에이전트가 스스로 작업 계획을 세우고 단계별로 실행하는 과정이 화면에 실시간으로 표시됩니다.

**5단계 — 결과 검토 후 활용**  
작업이 끝나면 반드시 결과를 검토합니다. 틀린 내용은 수정하고, 괜찮은 부분은 바로 업무에 가져가세요. AI 결과를 무조건 신뢰하지 않는 습관이 중요합니다.

## 자주 묻는 질문 (FAQ)

**Q. AgentGPT는 무료인가요?**  
A. 웹 버전 기본 기능은 무료로 체험할 수 있습니다. 더 많이, 더 복잡하게 쓰려면 OpenAI API 키 연결이 필요하고, 사용량에 따라 API 비용이 발생합니다.

**Q. 회사 내부 자료를 입력해도 안전한가요?**  
A. 민감한 고객 정보나 재무 데이터는 넣지 않는 게 원칙입니다. 공개해도 무방한 수준의 자료만 활용하세요. 보안이 중요한 환경이라면 GitHub에서 소스코드를 받아 자체 서버에 설치하는 방식을 고려해야 합니다.

**Q. ChatGPT가 있으면 AgentGPT가 필요 없지 않나요?**  
A. ChatGPT는 질문 하나에 답 하나를 주는 데 강합니다. AgentGPT는 여러 단계의 작업을 자동으로 이어가는 데 강점이 있습니다. "경쟁사 10곳 조사 → 비교표 작성 → 요약 보고서 초안"처럼 연속 작업이 필요할 때 AgentGPT가 빛납니다.
