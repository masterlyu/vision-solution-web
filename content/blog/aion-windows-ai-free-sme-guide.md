사무실 컴퓨터에서 ChatGPT를 쓰다가 멈칫한 적 있으세요?

"이 계약서 내용을 붙여넣어도 되나…" "거래처 정보가 미국 서버로 넘어가는 거 아닌가…"

이 불안함은 사실입니다. 클라우드 AI는 내가 입력한 내용을 외부 서버로 전송합니다. 중요한 기업 정보를 다룰 때는 신중할 수밖에 없습니다.

마이크로소프트가 2026년 6월 2일 Build 행사에서 발표한 **Aion 1.0**은 이 문제를 정면으로 해결합니다. 인터넷도, 구독료도, 외부 서버도 필요 없습니다. AI가 내 윈도우 컴퓨터 안에서만 돌아갑니다.

## Aion 1.0이 뭔가요?

**Aion 1.0은 윈도우 11에 직접 내장되는 AI 모델입니다.**

지금까지 AI를 쓰려면 ChatGPT, Claude, Gemini 같은 서비스에 로그인해야 했습니다. 내가 입력한 내용이 인터넷을 통해 해외 서버로 이동하고, 거기서 처리된 결과가 다시 내 화면으로 돌아오는 구조입니다.

Aion은 다릅니다. **모델 자체가 내 PC 안에 설치**됩니다. 내가 입력한 내용은 인터넷 밖으로 나가지 않습니다. 처음부터 끝까지 내 컴퓨터 안에서만 처리됩니다.

마이크로소프트는 이것을 "unmetered intelligence(무제한 지능)"라고 불렀습니다. 클라우드 AI처럼 사용할 때마다 비용이 나가지 않는다는 뜻입니다.

---

## 모델이 두 가지입니다

![Aion 1.0 두 모델 비교](/images/blog/aion-windows-ai-free-sme-guide-fig1.svg)
*▲ Aion Instruct(일상 업무)와 Aion Plan(자율 에이전트) · 출처: 마이크로소프트 Build 2026*

### Aion 1.0 Instruct — 일상 업무용

문서 요약, 이메일 초안 작성, 문구 다듬기에 최적화된 모델입니다. Edge 브라우저에 통합되어, 지금 당장 Edge Insider(베타 채널)에서 미리 체험할 수 있습니다. 2026년 7월에는 Hugging Face에 오픈 웨이트(무료 다운로드 모델)로도 공개 예정입니다.

### Aion 1.0 Plan — 자율 에이전트용

140억 개(14B) 파라미터를 가진 추론·에이전트 모델입니다. 파일을 열고, 작업 순서를 계획하고, 여러 단계를 알아서 처리합니다. 예를 들어 "이달 견적서 폴더에서 금액 합산해서 보고서 만들어줘"처럼 복잡한 지시도 소화합니다.

단, Plan 모델은 **40 TOPS 이상의 NPU**가 탑재된 PC에서 구동됩니다. 최신 인텔·퀄컴 기반 코파일럿+ PC라면 대부분 해당됩니다. 기존 PC에서는 Instruct 모델부터 시작하면 됩니다.

## 지금 당장 써볼 수 있는 방법

### 방법 1: Edge Insider(베타)로 지금 체험

1. **Microsoft Edge Insider** 페이지에서 Beta 채널 다운로드
2. Edge 우측 상단 사이드바 아이콘 → AI 어시스턴트 열기
3. 웹페이지 내용 요약, 이메일 초안 작성 등 Instruct 기능 바로 사용 가능

인터넷은 연결되어 있어야 하지만, Edge Insider를 통해 Aion Instruct의 성능을 미리 맛볼 수 있습니다.

### 방법 2: Ollama로 유사 환경 지금 구축

Aion Plan의 정식 출시 전까지, 같은 '내 PC 안에서만 도는 AI' 경험을 원하신다면 **Ollama**가 대안입니다.

```
ollama pull gemma3:12b
```

터미널(명령 프롬프트)에 위 한 줄을 실행하면 구글의 12B 모델이 내 PC에 설치됩니다. 인터넷 연결 없이 문서 요약, 이메일 작성을 처리할 수 있습니다.

## 중소기업 실전 활용 3가지

### 1. 거래처 계약서 안심 검토

거래처에서 받은 계약서 PDF를 AI에게 검토 맡기기. 클라우드 AI라면 회사 기밀이 외부 서버로 이동합니다. Aion(또는 Ollama)이라면 내 PC 안에서만 처리됩니다.

> "이 계약서에서 위약금 조항과 자동연장 조항만 뽑아서 요약해줘"

계약서 텍스트를 붙여넣고 이 한 줄만 입력하면 됩니다.

### 2. 고객 이메일 초안 5초 완성

바이어나 거래처에서 영어 이메일이 왔을 때, 번역하고 적절한 답변을 쓰는 데 시간이 많이 걸립니다. AI에게 맡기면 30초 안에 초안이 나옵니다. 중요한 거래 내용이 외부로 유출될 걱정 없이요.

> "아래 영어 이메일에 대한 정중한 한국어 답변 초안을 써줘. 납기는 3주라고 답해."

### 3. 폴더 정리 자동화 (Plan 모델)

Aion Plan 모델이 정식 탑재되면 이런 지시가 가능해집니다.

> "Downloads 폴더에서 올해 계약서 파일만 골라서 '2026_계약서' 폴더에 옮겨줘"

에이전트 모델이 파일 목록을 스캔하고, 날짜와 파일명을 분석해서 직접 이동까지 처리합니다. 반나절 걸리던 정리 작업이 몇 초로 줄어듭니다.

## 비교: 클라우드 AI vs Aion 1.0

| 항목 | ChatGPT·Claude(클라우드) | Aion 1.0(윈도우 내장) |
|------|--------------------------|----------------------|
| 데이터 위치 | 외부 서버(해외) | 내 PC 안 |
| 월 비용 | 2만~3만원/월 | 무료(윈도우 포함) |
| 인터넷 필요 | 항상 필요 | 불필요(오프라인 사용) |
| 최신 정보 검색 | 가능 | 불가(학습된 데이터 기준) |
| 파일 직접 조작 | 제한적 | Plan 모델로 가능 |
| 현재 사용 가능 | 지금 바로 | Instruct: Edge Insider / Plan: 출시 예정 |

클라우드 AI와 로컬 AI는 경쟁 관계가 아닙니다. 민감한 내부 문서는 Aion으로, 최신 정보 검색이나 복잡한 창작 작업은 클라우드 AI로 — 목적에 따라 나눠 쓰는 게 가장 똑똑한 방법입니다.

## 언제 내 PC에 들어오나요?

| 일정 | 내용 |
|------|------|
| 2026년 6월 (지금) | Edge Insider Beta에서 Instruct 미리 체험 가능 |
| 2026년 7월 예정 | Aion Instruct 오픈 웨이트 공개(Hugging Face) |
| 2026년 하반기 예정 | 코파일럿+ PC에 Aion Plan 정식 내장 |

NPU 40 TOPS 이상의 새 PC를 구매할 계획이 있다면, Aion Plan이 탑재되는 시점과 맞춰 고려해 보세요. 구형 PC라면 Ollama로 지금 바로 비슷한 경험을 시작할 수 있습니다.

---

클라우드 AI가 편하지만 "회사 정보를 넣어도 되나?"라는 불안감이 있었다면, Aion 1.0이 그 불안을 없애줄 수 있습니다. 우리 회사 데이터는 우리 컴퓨터 안에서만 처리되는 AI — 마이크로소프트가 이걸 윈도우에 기본 탑재하는 방향으로 가고 있습니다.

지금은 Edge Insider와 Ollama로 먼저 경험해 보고, 정식 출시를 기다리는 것이 현명한 순서입니다.

> 우리 회사 PC에 Aion을 써볼 수 있는지, AI 도입 상담이 필요하다면 언제든지 문의해 주세요.
