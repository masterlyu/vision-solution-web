## 도입 전 상황

강원도에서 건강식품을 판매하는 ○○쇼핑몰은 직원 3명이 운영하는 소규모 회사입니다. 하루 평균 CS 문의가 50건이었고, 이 중 2명의 직원이 거의 풀타임으로 전화와 카카오톡 응대에 매달려 있었습니다.

**문제의 핵심**: 50건 중 40건은 반복적인 질문이었습니다.

- "주문했는데 언제 도착하나요?" (30%)
- "반품/교환 어떻게 하나요?" (25%)
- "할인 쿠폰은 어디서 받아요?" (15%)
- "성분에 OO이 들어있나요?" (10%)
- 기타 반복 질문 (20%)

## AI 챗봇 도입 결정

저희가 제안한 것은 고가의 서비스가 아니었습니다. Claude API + 쇼핑몰 데이터 연동으로 구축한 **맞춤형 CS 챗봇**입니다.

**구성 요소:**
1. 주문 상태 자동 조회 (API 연동)
2. 반품/교환 정책 문서 학습
3. 제품 상세 정보 데이터베이스
4. 카카오 채널 연동

## 도입 비용 공개

| 항목 | 비용 |
|------|------|
| 초기 구축비 | 280만원 |
| 월 API 사용료 (Claude) | 약 15만원 |
| 월 유지보수 | 10만원 |
| **월 총 비용** | **약 25만원** |

## ROI 계산

도입 전 CS 인건비:
- 직원 2명 × 30% 시간 = 0.6 FTE
- 최저임금 기준: 약 **120만원/월**

도입 후:
- AI가 80% 자동 처리 → 1명이 20% 시간만 사용
- 절감 인건비: **약 96만원/월**

**월 순절감: 71만원 → 초기 투자 회수 기간: 약 4개월**

## 실제 성과 (6개월 후)

- 자동 처리율: **82%**
- 고객 만족도: 3.2점 → 4.1점 (5점 만점)
- 직원 만족도: "이제 진짜 중요한 일에 집중할 수 있어요"
- 야간/주말 문의 처리: 100% 자동화

## 중요한 포인트

AI 챗봇은 만능이 아닙니다. 복잡한 불만, 감정적 대응이 필요한 상황은 여전히 사람이 처리합니다. 하지만 **단순 반복 업무를 AI에게 넘기는 것**만으로도 직원들이 고객 경험을 향상시키는 일에 집중할 수 있게 됩니다.
