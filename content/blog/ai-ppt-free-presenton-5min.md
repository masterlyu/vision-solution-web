사실 내용은 이미 다 알고 있었습니다. 정리도 됐고요. 문제는 "예쁘게 만드는 것" 자체에 너무 많은 시간이 들었다는 겁니다.

그런데 같은 내용을 AI에게 맡겼더니 5분이 걸렸습니다.

## Gamma·Beautiful AI, 쓰고 싶은데 매달 돈이 부담입니다

AI 발표 도구의 대명사는 **Gamma**입니다. 주제를 입력하면 슬라이드 초안이 뚝딱 나오는 서비스인데요. 써본 분들 반응이 대체로 좋습니다. 문제는 가격이에요.

Gamma Pro는 월 20달러(약 2만 7천 원)입니다. 매달 내야 하는 구독료입니다. **Beautiful AI**는 더 비싸요. 월 45달러(약 6만 원)를 내야 합니다. 팀원 5명이 다 써야 한다면 매달 몇십만 원이 나갑니다.

"한 달에 발표 자료 두어 번 만드는데 이 돈이 맞나?" 싶은 분들이 많습니다. 중소기업 예산으로는 결정이 쉽지 않죠.

그런데 같은 기능을 **무료**로 쓸 수 있는 오픈소스가 있습니다. GitHub에서 별을 8천 개 넘게 받은 도구인데요, 이름이 **Presenton**입니다.

---

## Presenton이 뭔가요? — 쉽게 설명하면

**오픈소스**란 "누구나 무료로 쓸 수 있는 프로그램"입니다. Presenton은 이런 무료 AI 발표 생성기예요. GitHub(github.com/presenton/presenton)에 공개돼 있고, Apache 2.0 라이선스라 상업적으로도 무료 사용이 가능합니다.

작동 방식은 간단합니다.

1. 주제를 한 줄 입력합니다 ("2026년 하반기 영업 전략 보고서")
2. AI가 슬라이드 구성을 잡고 내용을 채웁니다
3. PPTX로 다운로드해 파워포인트에서 바로 수정합니다

![center](/mascot/md/service/cat_svc_ai.png)

이 과정에 걸리는 시간이 5분 안팎입니다. 디자인도 자동으로 잡아주고, 이미지도 알아서 붙여줍니다.

좋은 점만 늘어놓는 건 의미가 없습니다. 실제로 어떤지 보겠습니다.

**Gamma가 Presenton보다 나은 점:** 설치 없이 브라우저에서 바로 사용, 팀 협업이 더 편리, UI가 더 세련됨.

결론적으로 — 발표 자료를 자주 만들고, 비용을 줄이고 싶고, 회사 데이터 보안이 중요한 분께는 Presenton이 맞습니다. 반면 당장 오늘 쓰고 싶고 설치가 귀찮다면 Gamma 무료 플랜(월 1회 제한)이 더 빠른 선택입니다.

## 내 회사에선 어떻게 쓸 수 있을까요?

실제 업무에서 가장 많이 쓰이는 세 가지 시나리오입니다.

**시나리오 1 — 영업 제안서 (영업팀)**
거래처별 맞춤 제안서가 필요한데, 매번 처음부터 만들기엔 시간이 너무 걸립니다. Presenton에 제품 정보와 거래처 특성을 입력하면 10분 안에 초안이 완성됩니다. 거기서 로고와 수치만 수정하면 나만의 제안서가 됩니다.

**시나리오 2 — 월간 보고서 (경영지원팀)**
엑셀에 정리된 실적 데이터를 텍스트로 변환해 붙여넣으면, AI가 핵심 지표를 추려 슬라이드로 구성합니다. "지난달 대비 매출 12% 증가, 신규 고객 23명 유입" 같은 내용을 시각화된 보고서로 만드는 데 5분이면 충분합니다.

**시나리오 3 — 직원 교육 자료 (총무·HR팀)**
직원 입사 교육, 내규 안내, 시스템 사용법 등 반복적으로 만드는 교육 자료를 AI로 초안 잡아두면 다음엔 수정만 하면 됩니다. 한 번 만들어둔 PPTX 템플릿을 Presenton에 업로드하면 같은 디자인으로 새 자료를 계속 생성할 수 있습니다.

![center](/mascot/md/process/cat_plan.png)

> **(주)비젼솔루션 관점** — 발표 자료 제작 시간이 줄어드는 것보다 중요한 게 있습니다. 바로 "만드는 것"에서 "생각하는 것"으로 에너지가 이동한다는 점입니다. AI가 슬라이드 형식을 잡아주면, 담당자는 거래처 입장에서 어떤 순서로 설명해야 설득력이 높은지에 집중할 수 있습니다. 도구는 도구고, 핵심은 여전히 사람의 판단입니다. AI 발표 도구가 대체하는 건 "손으로 하는 작업"이지 "머리로 하는 전략"이 아닙니다.

## 지금 바로 실습해보기

두 가지 방법이 있습니다. 설치가 싫다면 웹 버전, 보안이 중요하다면 Docker 설치 버전을 고르세요.

### 방법 A — 웹 버전 (설치 없음, 가장 빠름)

**1단계 — 웹사이트 접속**

브라우저에서 [presenton.ai](https://presenton.ai) 를 엽니다. 화면 상단 "Try Now" 또는 다운로드 버튼을 클릭합니다.

**2단계 — AI 모델 연결**

OpenAI API 키가 있다면 입력합니다. (platform.openai.com에서 무료 발급 가능) Google Gemini API 키도 됩니다. Gemini는 일정 사용량까지 무료입니다.

**3단계 — 주제 입력**

아래처럼 입력해보세요:

```
2026년 하반기 마케팅 전략 보고서
우리 회사는 제조업, 직원 30명 규모
대상: 경영진 보고용, 15슬라이드
```

**4단계 — 생성 및 다운로드**

AI가 자동으로 구성을 잡고 슬라이드를 만듭니다. 완료 후 PPTX로 내려받아 파워포인트에서 수정하세요.

---

### 방법 B — Docker 설치 (데이터 보안 우선)

> Docker가 뭔지 모르는 분들: "프로그램을 컴퓨터에 깔아두는 방법 중 하나"라고 생각하시면 됩니다.

Docker가 없다면 먼저 [docker.com](https://docker.com)에서 무료로 설치합니다.

이후 아래 명령어 한 줄을 실행하면 Presenton이 내 컴퓨터에 설치됩니다:

```bash
docker run -it --name presenton -p 5000:80 -v "./app_data:/app_data" ghcr.io/presenton/presenton:latest
```

설치 후 브라우저에서 `http://localhost:5000` 을 열면 됩니다. 이후 과정은 웹 버전과 동일합니다.

**💡 컴퓨터 사양 참고:**
- 웹 버전 사용 시: 별도 사양 불필요 (클라우드 AI 모델 사용)
- Docker 자체 설치 + 클라우드 AI 모델(OpenAI 등): RAM 8GB 이상 권장
- Ollama(완전 오프라인 로컬 AI) 연결 시: RAM 16GB 이상, 고사양 그래픽카드(GPU) 권장

대부분의 회사 PC에서는 방법 B + OpenAI API 연결 조합이 현실적입니다.

## 자주 묻는 질문 (FAQ)

**Q. AI API 키 없이 쓸 수 있나요?**
A. 일부 기본 기능은 API 키 없이 체험 가능합니다. 제대로 쓰려면 OpenAI나 Gemini API 키가 필요합니다. OpenAI API는 처음 가입하면 무료 크레딧을 주는데, 발표 자료 5~10개 정도 만들어볼 수 있습니다.

**Q. 만들어진 PPT를 파워포인트에서 수정할 수 있나요?**
A. 네, PPTX 파일로 다운로드하면 일반 파워포인트에서 글자, 이미지, 색상 모두 수정 가능합니다.

**Q. 회사 로고와 색상을 반영할 수 있나요?**
A. 기존 회사 PPT 파일을 Presenton에 업로드하면 그 디자인을 기억해서 새 자료에 적용합니다.

**Q. 한국어로 발표 자료 만들어도 자연스럽나요?**
A. 대체로 자연스럽습니다. 다만 특수 전문 용어나 업계 은어는 영어로 나오거나 어색할 수 있어, 완성 후 한 번 검토하는 게 좋습니다.
