## 20분 설치 가이드 — 3단계

Docker가 설치된 Ubuntu 서버가 있다면 아래 순서로 진행합니다.

![Postiz Docker 설치 3단계](/images/blog/postiz-sns-ai-scheduler-free-guide-fig2.svg)
*▲ Postiz Docker Compose 설치 흐름 · 출처: docs.postiz.com*

**1단계 — 저장소 내려받기**

```bash
git clone https://github.com/gitroomhq/postiz-docker-compose
cd postiz-docker-compose
```

**2단계 — 환경 설정**

`postiz.env` 파일을 열고 아래 두 줄을 내 서버 IP로 채웁니다.

```
MAIN_URL=http://내서버IP:4007
NEXT_PUBLIC_BACKEND_URL=http://내서버IP:3000
```

AI 글쓰기 기능을 쓰려면 OpenAI 또는 Anthropic API 키를 추가로 입력합니다(사용량에 따라 소액 발생).

**3단계 — 실행**

```bash
docker compose up -d
```

완료되면 브라우저에서 `http://내서버IP:4007`에 접속합니다. 계정을 만들고 SNS 채널을 연결하면 바로 쓸 수 있습니다.

> ⚠️ AI 이미지·영상 생성은 별도 API 키가 필요합니다. **텍스트 초안 생성과 예약 발행은 API 키 없이도 사용 가능합니다.**

## 자주 묻는 질문

**Q. 서버가 없어도 Postiz를 쓸 수 있나요?**  
postiz.com에서 클라우드 플랜($29~/월)을 구독하면 설치 없이 바로 사용합니다. 채널이 3개 이하라면 설치보다 클라우드 플랜이 더 간편할 수 있습니다.

**Q. 인스타그램·유튜브 연결이 정말 되나요?**  
Instagram Business 계정·YouTube 채널·Facebook 페이지 등 공식 API를 지원하는 채널은 연동됩니다. 단, Instagram은 비즈니스 계정이 필수이며 Facebook App 설정이 필요합니다.

**Q. AI 글쓰기는 한국어도 되나요?**  
OpenAI나 Anthropic API 키를 연결하면 한국어 초안도 작성됩니다. "인스타 캡션 한국어로 써줘" 식으로 입력하면 됩니다.
