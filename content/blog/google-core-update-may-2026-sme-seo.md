## 지난주부터 방문자가 줄었다면, 이것부터 확인하세요

혹시 최근 홈페이지 방문자 수가 평소보다 줄어든 것 같다는 느낌이 드시나요? 착각이 아닐 수 있습니다.

구글은 **2026년 5월 21일부터 5월 코어 업데이트(May 2026 Core Update)를 공식 시작**했습니다. 구글은 [Google Search Central 코어 업데이트 공식 페이지](https://developers.google.com/search/updates/core-updates)를 통해 코어 업데이트 현황을 공개하며, 이번 업데이트는 검색 순위 평가 기준 자체를 대규모로 재정비하는 변경으로 보통 2~4주에 걸쳐 순차 적용됩니다.

SEO 전문 미디어 [랭크온](https://akeep.co.kr/seo-optimization-2026/)이 2026년 5월에 발표한 자료에 따르면, 이번 업데이트에서 **기본 SEO 구조가 미비한 사이트는 검색 순위가 하락**하는 패턴이 나타나고 있습니다.

대기업은 전담 SEO 팀이 있어 빠르게 대응하지만, 중소기업 홈페이지는 처음 만들어 놓고 그대로 방치된 경우가 많습니다. 지금 이 글을 읽고 계신다면, 딱 10분만 투자해서 우리 홈페이지 상태를 점검해 보세요.

## 구글 코어 업데이트, 어떤 홈페이지가 타격받나요?

### 기술적 구조 문제

랭크온 자료에서 이번 업데이트에 취약하다고 명시한 사이트 유형은 다음과 같습니다:

- **페이지 로딩이 느린 사이트**: 모바일에서 3초 이상 걸리면 위험
- **Core Web Vitals 기준 미달**: LCP(최대 콘텐츠 렌더링 속도), CLS(화면 레이아웃 안정성), INP(클릭 반응성) 세 가지 지표가 모두 중요
- **모바일 최적화 미비**: 2026년 기준 모바일 트래픽이 전체의 70%를 넘었습니다 ([sowonweb.co.kr, 2026년 홈페이지 제작 트렌드 총정리](https://sowonweb.co.kr/blog/homepage-design-trends-2026/))
- **구조화 데이터 누락**: Schema.org 마크업이 없으면 구글이 페이지 내용을 정확히 이해하지 못합니다

### 콘텐츠 문제

- 제품·서비스 설명만 나열하고 고객에게 실질적인 정보를 제공하지 않는 페이지
- 메타 제목(title)과 메타 설명(description)이 비어 있거나 중복된 페이지
- 마지막 업데이트가 1년 이상 된 콘텐츠

위 항목 중 하나라도 해당된다면, 이번 업데이트로 이미 순위가 내려갔거나 앞으로 내려갈 가능성이 높습니다.

홈페이지 구조가 걱정된다면 [홈페이지 리뉴얼 서비스](/renewal)를 통해 속도·SEO·모바일 최적화를 한번에 개선할 수 있습니다.

## SEO만으로는 부족한 시대 — AEO가 뭔가요?

2026년에는 SEO(검색엔진 최적화) 하나만으로는 완전하지 않습니다. 네이버 AI 검색, 구글 AI Overviews 같은 기능이 강화되면서 사용자들이 검색 결과 목록을 보는 대신 **AI가 요약해주는 답변에서 바로 정보를 얻는 경우**가 늘고 있습니다.

이에 따라 **AEO(Answer Engine Optimization, AI 답변 엔진 최적화)** 가 2026년의 핵심 전략으로 부상했습니다.

[픽셀아트웍스(pixelart-works.com)](https://pixelart-works.com/Column/?bmode=view&idx=169357981)의 분석에 따르면, AI 답변에 우리 브랜드 홈페이지가 노출되려면 다음이 필요합니다:

- **FAQ 구조**: "이 서비스는 어떤 회사에 적합한가요?" 같은 질문-답변 형식의 콘텐츠
- **명확하고 구체적인 단락**: AI가 발췌해서 답변으로 쓸 수 있는 완결된 문장
- **`llms.txt` 파일**: ChatGPT·Claude·Perplexity 등 LLM 크롤러가 우리 사이트를 학습할 수 있도록 안내하는 텍스트 파일

특히 `llms.txt`는 2026년 테크니컬 SEO의 새로운 기술 표준으로 자리 잡고 있습니다 ([imweb.me, AI가 검색을 대체할까? 생성형 AI 시대 SEO·GEO 마케팅 총정리](https://imweb.me/blog?idx=432)). 아직 이 파일이 없는 홈페이지라면 AI 크롤러가 우리 사이트를 제대로 읽지 못하는 상황입니다.

## 지금 당장 확인할 수 있는 SEO·AEO 체크리스트

### ✅ 기술적 SEO 점검 (무료 도구로 가능)

| 점검 항목 | 확인 방법 |
|-----------|-----------|
| 페이지 속도 | Google PageSpeed Insights에서 모바일 점수 확인 (70점 이상 목표) |
| 모바일 최적화 | Google Mobile-Friendly Test 통과 여부 확인 |
| HTTPS 보안 | 주소창 자물쇠 아이콘 표시 여부 (없으면 구글이 '안전하지 않은 사이트'로 분류) |
| Core Web Vitals | Google Search Console > 핵심 웹 지표 보고서 |
| 구조화 데이터 | Google Rich Results Test에서 Schema 마크업 확인 |

### ✅ 콘텐츠 SEO 점검

- 각 페이지에 고유한 H1 제목이 있는가
- 페이지마다 메타 디스크립션(요약 설명)이 작성되어 있는가
- 고객이 실제로 검색할 키워드가 자연스럽게 포함되어 있는가
- 최근 3개월 이내에 새 콘텐츠(블로그, 공지 등)가 추가되었는가

### ✅ AEO (AI 답변 최적화) 점검

- 서비스 페이지에 FAQ 섹션이 있는가
- 주요 서비스 설명이 질문-답변 형식으로 구성되어 있는가
- 루트 도메인(예: yourdomain.com/llms.txt)에 `llms.txt` 파일이 있는가

이 중 절반 이상에서 '아니오'가 나온다면, 이번 구글 업데이트 이후 우리 홈페이지가 경쟁사보다 뒤처지고 있을 가능성이 높습니다.

현재 홈페이지 상태가 정확히 어떤지 파악하고 싶다면 [무료 보안·SEO 진단](/security)을 신청해 보세요. 전문가가 직접 점검 후 보고서를 드립니다.
