## 마케팅팀 없이 홍보영상을 만들 수 있다면?

영상 1편에 외주비 50만~200만 원. 중소기업이 감당하기엔 버거운 금액입니다. 그렇다고 직원이 직접 편집하자니 시간도 기술도 없죠.

이번 주 GitHub(개발자들이 코드를 공유하는 세계 최대 오픈소스 플랫폼)에서 하루 별 3,000개를 받으며 트렌딩 2위에 오른 도구가 있습니다. 이름은 **OpenMontage**. "AI 에이전트가 주제 입력 한 줄로 영상을 처음부터 끝까지 만들어준다"는 게 핵심입니다.

이 도구가 정확히 뭔지, 실제로 쓸 수 있는지, 중소기업에 어떻게 접목하는지 차례로 살펴보겠습니다.

## OpenMontage란? — '에이전트 AI'를 쉽게 이해하는 법

OpenMontage는 오픈소스 **에이전트형** 영상 제작 시스템입니다. '에이전트 AI'란 쉽게 말해 "스스로 계획 세우고 실행하는 AI"입니다. 기존 AI가 질문에 답하는 수준이라면, 에이전트는 목표를 주면 알아서 단계를 밟아 결과물을 냅니다.

OpenMontage의 자동화 흐름은 이렇습니다:

**① 리서치** → 주제에 맞는 정보·자료를 AI가 스스로 수집  
**② 스크립트 작성** → 수집 내용으로 영상 대본을 자동 작성  
**③ 에셋 생성** → 이미지·배경음악·나레이션을 AI가 직접 생성  
**④ 편집** → 클립 배치·자막·효과를 자동 적용  
**⑤ 최종 렌더링** → 완성된 영상 파일 출력

12개 파이프라인, 52개 도구, 400개 이상의 에이전트 스킬이 협력해 돌아가는 구조입니다. Claude Code·Cursor 같은 AI 개발 도구와도 연동됩니다.

## 실제로 쓸 수 있을까? — 현실 요구사항과 솔직한 장단점

![center](/mascot/md/emotion/cat_thinking.png)

"좋다"는 말보다 중요한 건 "내 PC에서 돌아가냐"는 질문입니다.

**컴퓨터 사양 (일반적 권장 기준):**

| 항목 | 최소 | 권장 |
|------|------|------|
| RAM | 16GB | 32GB 이상 |
| GPU | 없어도 가능 (속도 느림) | NVIDIA GPU 8GB 이상 |
| 저장공간 | 50GB 여유 | 100GB 이상 |
| 인터넷 연결 | 필수 (AI API 연동) | 고속 권장 |

*공식 README에 하드웨어 최소 사양 명시 없음. 위 수치는 유사 AI 영상 도구 운용 경험 기반 일반 권장값입니다.*

**솔직한 장단점:**

| 장점 | 단점 |
|------|------|
| 무료·오픈소스 (AGPLv3 라이선스¹) | 초기 설치에 개발 지식 필요 |
| 전 과정 완전 자동화 | 세밀한 수작업보다 품질이 낮을 수 있음 |
| Claude Code·Cursor 연동 지원 | API 사용 시 별도 비용 발생 가능 |
| 커스텀·확장 자유로움 | GPU 없으면 렌더링 30분 이상 걸릴 수 있음 |

**대안 비교:**
- **Synthesia·HeyGen(클라우드)**: 설치 없이 바로 사용·깔끔한 UI. 단, 월 구독료 $30~300 수준.
- **OpenMontage**: 무료이지만 초기 설치 장벽 존재.
- **Runway·Kling AI**: 클립 단위 생성에 강하지만, 전 과정 자동화는 약함.

GPU가 없고 기술 지식이 전무하다면, 처음엔 클라우드 서비스로 시작한 뒤 비용 절감이 필요한 시점에 OpenMontage를 검토하는 순서가 현실적입니다.

## 중소기업 활용 시나리오 — 실제로 어떻게 쓰나요?

![center](/mascot/md/emotion/cat_surprised.png)

**시나리오: 소규모 식품 제조업체 K사**

K사 마케팅 담당자가 신제품 홍보영상을 만들어야 합니다. 외주 견적은 120만 원, 납기는 2주. OpenMontage를 활용하면:

1. 담당자가 프롬프트 입력:  
   *"제품: 국산 참깨 착유 들기름. 타겟: 40~60대 주부. 소구점: 100% 국산·저온압착. 30초 홍보영상 만들어줘."*
2. AI가 경쟁 제품 자동 리서치 → 강조 포인트 정리 → 대본 작성
3. 배경음악·나레이션·제품 이미지 자동 생성 후 편집
4. 30초 완성 영상 출력 (소요 시간: 약 15~30분)

비용은 전기값·인터넷 이외 거의 0원. 외주 기다리는 2주 대신 30분 안에 시안이 나옵니다.

**회사 데이터 연결 방법:**  
제품 소개서(PDF)나 브랜드 가이드를 프롬프트에 함께 제공하면 AI가 그 내용을 반영해 대본을 씁니다. 실제 제품 사진을 넣으려면 이미지 파일을 에셋 폴더에 추가하면 됩니다.

**반복 자동화:**  
신제품이 나올 때마다 같은 프롬프트 구조를 재사용하거나, 크론 스케줄러로 정기 자동 실행도 가능합니다. 매달 영상 1편을 수동 외주 없이 뽑을 수 있는 구조가 됩니다.

## 직접 해보기 — 초보자 실습 5단계

> **사전 준비:** Python 3.10 이상, Git이 설치된 PC 또는 노트북. (각 공식 사이트에서 무료 설치 가능)

**① 프로젝트 다운로드**
```bash
git clone https://github.com/calesthio/OpenMontage
cd OpenMontage
```

**② 패키지 설치**
```bash
pip install -r requirements.txt
```

**③ API 키 설정 (`.env` 파일 생성)**
```
ANTHROPIC_API_KEY=여기에_본인_Claude_API_키
```

**④ 실행**
```bash
python main.py
```

**⑤ 프롬프트 입력**
```
주제를 입력하세요: 30초 카페 홍보 영상, 따뜻한 감성, 소규모 카페
```

AI가 자동으로 파이프라인을 실행하며 영상을 만들기 시작합니다.

*첫 실행은 모델 파일 다운로드로 수 분이 걸릴 수 있습니다. 이후 실행부터는 훨씬 빠릅니다.*
