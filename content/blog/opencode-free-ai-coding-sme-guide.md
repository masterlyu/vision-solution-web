## AI 코딩 에이전트가 뭔가요?

AI 코딩 에이전트란, 쉽게 말하면 **내 컴퓨터 파일을 직접 열고, 코드를 수정하고, 저장까지 해주는 AI 비서**입니다.

ChatGPT에 "이 코드 고쳐줘"라고 물으면 답이 채팅창에 나옵니다. 복사해서 파일에 붙여넣는 건 직접 해야 하죠. AI 코딩 에이전트는 그 귀찮은 과정을 건너뜁니다. AI가 파일을 직접 열고, 수정하고, 저장까지 합니다.

비유하자면, 지시만 하면 알아서 일하는 개발 보조 직원입니다. "메인 페이지 배너 문구를 '여름 특가 30%'로 바꿔줘"라고 말하면 AI가 HTML 파일을 찾아 직접 고쳐줍니다.

Cursor, GitHub Copilot, Windsurf 같은 도구가 이 범주인데, 대부분 월 2만~4만원 구독이 필요합니다. OpenCode는 같은 일을 **완전 무료 오픈소스**로 합니다.

2026년 6월 기준 OpenCode의 현황입니다.

**지원 AI 모델: 75개 이상**  
Claude, GPT-4o, Gemini, 그리고 내 컴퓨터 안에서만 돌아가는 Ollama까지 연결할 수 있습니다. AI 모델을 입맛대로 고를 수 있다는 뜻입니다.

**완전 무료 사용: Ollama 연동**  
인터넷 없이 내 컴퓨터 안에서만 AI를 돌리는 방식이 있습니다. Ollama를 설치하면 됩니다. 이 경우 API 비용이 전혀 들지 않습니다. 속도는 PC 사양에 따라 조금 느려질 수 있지만, 월 비용은 $0 그대로입니다.

**솔직한 한계도 있습니다**

## 중소기업이 실제로 쓰는 방법 — 시나리오 3가지

홈페이지 소스 코드가 있는 폴더에서 OpenCode를 실행하면, AI가 해당 폴더 안 파일을 읽고 수정 제안을 합니다. 자주 쓰이는 시나리오를 보겠습니다.

**시나리오 1 — 홈페이지 텍스트·배너 수정**

Before: 이벤트 배너 문구를 바꾸려면 개발자한테 카카오톡 → 2~3일 대기 → 수정 확인 → 비용 청구  
After: `"메인 배너 제목을 '여름 특가 30% 할인'으로 바꿔줘"` 입력 → AI가 HTML 파일 찾아 수정 → 5분 완료

**시나리오 2 — 엑셀 업무 자동화**

"이 엑셀 데이터를 자동으로 정리해주는 Python 스크립트 만들어줘"라고 지시하면, AI가 스크립트를 직접 작성하고 저장합니다. 반복 작업을 한 번에 끝낼 수 있습니다.

**시나리오 3 — 내 데이터 연결**

업무 문서가 있는 폴더에서 OpenCode를 실행하면, AI가 해당 폴더 파일을 읽고 분석해줍니다. 계약서 요약, 공지 초안 작성 등 문서 작업에도 활용 가능합니다.

단, 카페24·아임웹 같은 빌더 기반 홈페이지는 소스 파일에 직접 접근이 안 돼 파일 수정은 어렵습니다. 이 경우 "어떻게 수정해야 하는지 방법을 알려달라"는 용도로 활용하면 됩니다.

깊은 커스텀이나 기존 시스템 연동은 "여기까지는 직접, 그 이상은 전문가와"로 선을 명확히 하는 게 시간과 비용을 아낍니다.

## 지금 바로 따라하는 5분 설치

![OpenCode 시작 3단계](/images/blog/opencode-free-ai-coding-sme-guide-fig2.svg)
*▲ OpenCode 설치·실행 흐름 · 출처: opencode.ai 공식 설치 가이드*

Mac 또는 Linux 환경 기준입니다. (Windows는 WSL 설치 후 동일하게 진행)

**1단계 — OpenCode 설치**

터미널(Mac: Spotlight에서 "terminal" 검색)을 열고 아래 명령어를 붙여넣습니다.

```bash
curl -fsSL https://opencode.ai/install | bash
```

설치가 완료되면 `opencode --version`으로 정상 설치를 확인하세요.

**2단계 — AI 모델 연결**

*완전 무료로 쓰고 싶다면 — Ollama 연동*

[https://ollama.ai](https://ollama.ai)에서 Ollama를 설치한 뒤:

```bash
ollama pull llama3.2
```

OpenCode 실행 시 모델로 `ollama/llama3.2`를 선택하면 API 비용 없이 사용 가능합니다.

*Claude·GPT-4o를 쓰고 싶다면 — API Key 방식*

각 서비스에서 API Key를 발급받아 OpenCode에 입력합니다. 소량 사용 기준 월 수백 원~수천 원 수준입니다.

**3단계 — 실행하고 지시하기**

수정할 파일이 있는 폴더에서 터미널을 열고:

```bash
opencode
```

채팅창처럼 AI에게 지시합니다.

```
> "현재 폴더의 index.html에서 대표 전화번호를 010-1234-5678로 바꿔줘"
> "이 Python 파일에서 에러가 나는 부분을 찾아서 고쳐줘"
> "이 폴더 안의 Word 파일들을 요약해서 하나의 문서로 정리해줘"
```

AI가 수정 내용을 보여주고, 확인하면 파일에 직접 반영합니다.

**시스템 요구사항**

| 환경 | 필요 사양 |
|------|---------|
| OpenCode 자체 | Mac, Linux, Windows(WSL) + 인터넷 |
| 클라우드 AI 모델 (Claude, GPT 등) | API Key 필요, 인터넷 연결 |
| Ollama 로컬 모델 | RAM 8GB 이상 권장 (모델 크기별 상이) |
