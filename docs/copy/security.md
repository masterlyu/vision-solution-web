### 보안 진단 FAQ

**Q1. 진단이 진짜 무료인가요? 나중에 비용이 청구되지 않나요?**
> 보안 진단 리포트는 완전 무료입니다. 리포트를 받은 후 취약점 개선 작업을 의뢰하실지는 전적으로 선택이며, 강요하지 않습니다.

**Q2. URL만 입력하면 되나요? 관리자 계정이나 서버 접근이 필요한가요?**
> 외부에서 접근 가능한 공개 보안 요소만 분석합니다. 관리자 계정, FTP, 서버 접근 정보는 전혀 필요하지 않습니다.

**Q3. 진단 중에 사이트가 느려지거나 오류가 생기지 않나요?**
> 비파괴적(Non-intrusive) 방식으로 분석하므로 사이트 운영에 전혀 영향을 주지 않습니다.

**Q4. 리포트에는 어떤 내용이 담기나요?**
> ① SSL/HTTPS 상태 ② 보안 헤더 6가지 ③ 민감 정보 노출 여부 ④ 관리자 접근 보안 ⑤ SEO 신뢰도 영향 ⑥ 페이지 속도 — 이 6가지 항목에 대한 등급과 개선 방법이 포함됩니다.

**Q5. 리포트를 받고 나서 어떻게 해야 하나요?**
> 리포트에 항목별 우선순위와 직접 조치 방법이 함께 제공됩니다. 직접 수정이 어렵다면 개선 작업을 의뢰할 수 있으며, 별도 견적을 드립니다.

**Q6. WordPress가 아닌 사이트도 진단되나요?**
> 네. WordPress, Cafe24, 자체 개발 사이트 등 플랫폼에 관계없이 URL 기반으로 분석합니다.

**Q7. 이미 보안이 잘 되어 있는 사이트도 진단하면 의미가 있나요?**
> 네. 진단 결과 "이상 없음"이 나오면 그것 자체가 증거입니다. 리포트를 고객 신뢰 자료로 활용하는 분들도 있습니다.

**Q8. 진단 후 연락이 오나요? 영업 연락이 오는 건 싫은데요.**
> 원하지 않으시면 연락드리지 않습니다. 이메일 주소를 입력하시면 리포트만 발송됩니다. 추가 연락 여부는 선택 사항입니다.

### 취약점 발견 시 개선 비용은?

> 진단은 무료이며, 발견된 취약점 개선 작업은 범위에 따라 별도 견적입니다.

| 작업 유형 | 가격 범위 | 내용 |
|---------|----------|------|
| 기본 보안 설정 | 30만원~ | SSL, 보안 헤더 6가지, 관리자 URL 변경 |
| 표준 취약점 개선 | 80만원~ | 기본 설정 + 플러그인 업데이트 + 권한 설정 |
| 심층 보안 강화 | 200만원~ | 전체 코드 취약점 스캔 + WAF 설치 + 모니터링 |

> 진단 리포트 기준으로 정확한 견적을 제공합니다.
