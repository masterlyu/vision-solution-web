- **모바일 퍼스트 설계** → 스마트폰으로 봐도 버튼이 크고 글씨가 선명합니다.
- **전환율 UX 재설계** → 방문자가 자연스럽게 "문의하기"를 누르게 됩니다.
- **SEO 구조 개편** → 구글·네이버에서 경쟁사보다 먼저 노출됩니다.

### 7. FAQ

**Q1. 리뉴얼하면 기존 고객 데이터나 SEO가 날아가지 않나요?**
> 기존 URL 구조를 최대한 유지하고, 301 리다이렉트로 검색 순위를 보전합니다. 고객 DB와 게시물 데이터는 마이그레이션 전 전수 백업 후 이전합니다.

**Q2. 기간이 얼마나 걸리나요?**
> 규모에 따라 다르지만, 일반 기업 홈페이지는 평균 4~5주입니다. 착수 전에 정확한 일정표를 드립니다. 일정 초과 시 사전에 반드시 말씀드립니다.

**Q3. 완성 후 수정이 필요하면 어떻게 하나요?**
> 납품 후 30일은 무상 수정을 제공합니다. 이후에는 유지보수 플랜(월 99,000원~)으로 계속 관리할 수 있습니다.

**Q4. 리뉴얼 중에도 기존 사이트를 계속 운영할 수 있나요?**
> 네. 별도 개발 환경에서 작업 후 완료 시 도메인만 전환하므로, 리뉴얼 기간 동안 기존 사이트는 정상 운영됩니다.

**Q5. 어떤 업종의 사이트를 많이 만드셨나요?**
> 제조업, 서비스업, 요식업, 병원/클리닉, IT 스타트업 등 다양한 업종을 경험했습니다. 업종별 전환율 높은 패턴을 적용합니다.

**Q6. 계약금은 얼마나 되나요?**
> 계약금 30%, 중도금 40%, 잔금 30% 구조입니다. 잔금은 완성물 확인 후 납부하시면 됩니다.
