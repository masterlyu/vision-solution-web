## 0. 디자인 방향

- **분위기**: 위기감(방치·해킹 공포) → 안심감(전담 파트너·24시간 케어)

    → 폰트: 1.05rem / 밝은 회색
  - CTA 버튼:
      "어떤 플랜이 맞는지 무료 상담받기 →"
      → primary 배경
