### 2. 공감 체크리스트

**혹시 이런 상황이세요?**

- [ ] 처음으로 홈페이지를 만들려는데 어디서 시작해야 할지 모르겠다
- [ ] 저렴한 곳에 맡겼다가 완성 후 연락이 안 돼서 고생한 경험이 있다
- [ ] 쇼핑몰인지, 브랜드 사이트인지, 어떤 유형이 맞는지 모르겠다
- [ ] 견적을 받으러 갔더니 용어가 어려워서 뭘 사는 건지 몰랐다
- [ ] 완성 후에도 글 하나 수정하려면 개발자에게 연락해야 한다
- [ ] 경쟁사에 비해 온라인 존재감이 너무 약하다고 느낀다

> 하나라도 해당된다면, 사이트 구조부터 다시 생각해야 합니다.

| 지표 | 현실 |

### 4. 솔루션 — 구축 후 이렇게 달라집니다

- **전환율 설계** → 방문자가 "여기다"라고 느끼고 문의 버튼을 누르게 됩니다.
- **모바일 퍼스트** → 스마트폰에서 먼저 완벽하게, PC에서도 깔끔하게.
- **직접 편집 가능** → 관리자 페이지 제공으로 글·이미지를 직접 수정할 수 있습니다.
- **SEO 기반 설계** → 구글·네이버 상위 노출을 처음부터 고려해서 구조를 잡습니다.

### 6. 진행 프로세스

```
STEP 01 — 요구사항 정의 (1~2일)
  목적·타겟·기능 목록 확정. 일정·예산 협의.
  "이런 게 필요할 것 같아요"부터 시작해도 됩니다.

STEP 02 — 기획·디자인 (1~2주)
  사이트맵, 와이어프레임, 디자인 시안 3종 제시.
  확정 전까지 수정 횟수 제한 없음.

STEP 03 — 개발 (2~4주)
  프론트엔드·백엔드 풀스택 + 관리자 페이지 개발.
  진행 상황은 매주 공유합니다.

STEP 04 — 배포·인수인계 (3일)
  서버 세팅, 도메인 연결, 운영 가이드 문서 제공.
```

> 전체 기간: **평균 5~8주** (유형·기능에 따라 상이)

### 8. FAQ

**Q1. 처음 맡기는 거라 막막한데, 어디서 시작하면 되나요?**
> 아이디어나 경쟁사 사이트 예시 하나만 있어도 충분합니다. 무료 상담에서 어떤 기능이 필요한지 함께 정리해 드립니다.

**Q2. 완성 후 직접 글과 사진을 수정할 수 있나요?**
> 네. 관리자 페이지를 제공해서 개발자 없이도 텍스트·이미지를 수정할 수 있습니다. 사용법은 영상 가이드로 제공합니다.

**Q3. 도메인과 서버(호스팅)도 대신 설정해주나요?**
> 네. 도메인 구매부터 서버 설정, SSL 인증서 설치까지 전부 대신해드립니다. 클라이언트분이 신경 쓸 부분이 없습니다.

**Q4. 완성 후 유지보수는 어떻게 되나요?**
> 납품 후 30일 무상 수정을 제공합니다. 이후에는 월 99,000원~의 유지보수 플랜으로 관리할 수 있습니다.

**Q5. 비슷한 업종 사이트 참고 사례를 볼 수 있나요?**
> 상담 신청 시 업종을 알려주시면 관련 포트폴리오를 보내드립니다.

**Q6. 중간에 기능 추가가 가능한가요?**
> 기획 확정 전까지는 변경이 자유롭습니다. 개발 착수 후 기능 추가는 추가 견적 협의가 필요하며, 사전에 안내드립니다.
