# 레이아웃 명세 — `/renewal` (홈페이지 리뉴얼)

> VIS-196 산출물 | AI 디자이너 작성 | 2026-04-17

---

## 0. 디자인 방향

- **분위기**: 긴장감(낡음/이탈) → 안도감(개선/회복) 여정
- **색상**: 기존 `--primary` / `--destructive` 조합 유지, 신규 색상 금지
- **타이포**: H1 강조(bold, 크기 크게), 본문은 간결하게
- **UI 원칙**: CTA 버튼 3곳 이상, 눈에 띄는 색상·크기(primary 배경 버튼)
- **모바일 퍼스트**: 모든 섹션 스택 레이아웃 기준 설계

## 2. 섹션별 상세 명세

### Section 1 — Hero

```
레이아웃: 2열 (텍스트 60% | 비주얼 40%)
배경: 다크 또는 딥 네이비 그라디언트

[좌측]
  - 배지(Badge): "홈페이지 리뉴얼" (primary 색상, 소형 pill)
  - H1 (2줄 이내):
      "지금 이 순간도,
       고객이 당신 사이트에서 떠나고 있습니다"
    → 폰트: 2.8rem / bold / 흰색
    → "떠나고" → --destructive 강조 색상
  - 서브카피 (2줄):
      "URL 하나만 입력하세요.
       48시간 내에 무료 진단 리포트를 보내드립니다."
    → 폰트: 1.1rem / 밝은 회색
  - CTA 버튼 (primary 배경, 크게):
      "지금 무료 진단 받기 →"
      → 패딩: 16px 32px / 라운드 8px

[우측]
  - Lottie/SVG: 이탈 화살표 + 속도계 아이콘 or 깨진 화면 일러스트
  - 크기: 400×350px (모바일: 숨김)
```

### Section 2 — 공감 체크리스트

```
레이아웃: 2열 카드 그리드 (모바일 1열)
배경: 밝은 회색 (#F8F9FA)

[섹션 헤더]
  - H2: "혹시 이런 상황이세요?" (1줄, 중앙 정렬)

[카드 구조 × 6]
  ┌─────────────────────────────────────┐
  │  [체크박스 아이콘 — 미체크, 회색]    │
  │  제목 텍스트 (1줄, 1rem bold)        │
  └─────────────────────────────────────┘
  카드: 흰 배경 / 그림자 shadow-sm / 라운드 8px
  패딩: 20px 24px
  항목:
    1. "홈페이지를 만든 지 3년이 넘었다"
    2. "모바일로 보면 레이아웃이 깨진다"
    3. "구글·네이버 검색이 잘 안 된다"
    4. "홈페이지는 있는데 문의가 거의 없다"
    5. "경쟁사 사이트와 비교해 초라해 보인다"
    6. "페이지 로딩이 느려서 고객이 떠날 것 같다"

[하단 결론 문구] (노란 경고 배경 or border-left --destructive 라인)
  "2개 이상 해당된다면, 홈페이지가 매출의 발목을 잡고 있습니다."
```

---

### Section 3 — 수치 근거

```
레이아웃: 4개 빅 넘버 카드 (수평, 모바일: 2×2)
배경: 흰색

[카드 구조 × 4]
  ┌────────────┐
  │  53%       │  ← 빅 넘버 (3rem bold, --primary 또는 --destructive)
  │  로딩 3초  │  ← 라벨 (0.85rem, 회색)
  │  이탈률    │
  └────────────┘

수치 4개:
  1. "53%" / 로딩 3초 초과 시 이탈률 급등
  2. "67%" / 모바일 미최적화 전환율 손실
  3. "2.3배" / 리뉴얼 후 문의 증가 (자사 기준)
  4. "140%" / SEO 개편 3개월 내 유입 증가
```

---

### Section 4 — 솔루션

```
레이아웃: 아이콘 + 텍스트 리스트 (좌측 아이콘, 우측 설명)
배경: 다크 (#1A1A2E 계열)

[섹션 헤더]
  - H2: "리뉴얼 후 이렇게 달라집니다" (흰색)

[리스트 × 4] — 각 행: 아이콘(32px) + 제목(bold) + 설명(1줄, 회색)
  ⚡ 속도 최적화        → 로딩 3초 → 0.8초. 고객이 기다리지 않습니다.
  📱 모바일 퍼스트       → 스마트폰에서도 버튼 크고 글씨 선명합니다.
  🎯 전환율 UX 재설계   → 방문자가 자연스럽게 "문의하기"를 누릅니다.
  🔍 SEO 구조 개편      → 경쟁사보다 먼저 검색 노출됩니다.
```

### Section 5 — 진행 프로세스

```
레이아웃: 수직 스텝퍼 (왼쪽 번호/라인, 오른쪽 내용)
배경: 밝은 회색

[스텝 × 4]
  STEP 01  무료 현황 진단    (2일)
  STEP 02  기획·설계         (1주)
  STEP 03  개발·이전         (2~3주)
  STEP 04  검수·배포         (3일)

스텝 카드 구조:
  - 번호: 원형 배지 (primary 배경, 흰색 텍스트)
  - 제목: bold 1rem
  - 기간: 회색 소형 배지
  - 연결선: 번호 사이 세로 점선

[하단 메모]
  "전체 기간: 평균 4~5주" — 회색 폰트, 오른쪽 정렬
```

### Section 6 — 가격표

```
레이아웃: 3열 카드 비교 (모바일: 세로 스택)
배경: 흰색

[섹션 헤더]
  "기본형 100만원대부터 시작합니다."
  서브: "정확한 견적은 무료 상담 후 제공합니다." (회색)

[카드 × 3]
  ┌──────────────────┐
  │  기본형           │
  │  100만원대~       │  ← 가격: 1.6rem bold
  │                   │
  │  • 디자인 리뉴얼   │
  │  • 모바일 최적화   │
  │  • 속도 개선       │
  └──────────────────┘

  카드 1: 기본형 — 일반 테두리
  카드 2: 표준형 — primary 테두리 + "추천" 배지 (primary 배경, 강조)
  카드 3: 확장형 — 일반 테두리

[CTA 2 버튼] (가격 섹션 하단)
  "무료 견적 확인하기 →" — primary 배경 버튼, 중앙 배치
```
