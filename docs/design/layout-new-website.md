# 레이아웃 명세 — `/new-website` (신규 사이트 구축)

> VIS-196 산출물 | AI 디자이너 작성 | 2026-04-17

---

## 0. 디자인 방향

- **분위기**: 막막함(처음 만드는 두려움) → 명확함(단계별 안내·전문가 파트너)
- **색상**: 기존 `--primary` / `--muted` 조합, 밝고 신뢰감 있는 톤
- **타이포**: 질문형 H1으로 공감 유도, 유형 카드는 아이콘 중심
- **특징**: 구축 유형 선택 카드 (브랜드/커머스/플랫폼/랜딩) — 시각적 차별화

---

## 1. 섹션 구조

| 순번 | 섹션 명 | 레이아웃 유형 |
|------|---------|--------------|
| 1 | Hero | 2열 (텍스트 + 일러스트) |
| 2 | 공감 체크리스트 | 2열 카드 그리드 |
| 3 | 수치 근거 | 4개 빅 넘버 카드 |
| 4 | 솔루션 | 2×2 결과 카드 |
| 5 | 구축 유형 | 4열 유형 선택 카드 |
| 6 | 진행 프로세스 | 수직 스텝퍼 |
| 7 | 가격표 | 3단 카드 비교 |
| 8 | FAQ | 아코디언 6개 |
| 9 | 하단 CTA | 풀 폭 배너 |

## 2. 섹션별 상세 명세

### Section 1 — Hero

```
레이아웃: 2열 (텍스트 55% | 비주얼 45%)
배경: 밝은 화이트 + 우측 primary 연한 그라디언트

[좌측]
  - 배지: "신규 홈페이지 구축" (primary 테두리 pill)
  - H1 (1줄):
      "홈페이지를 만들었는데 왜 문의가 없을까요?"
    → 폰트: 2.6rem / bold / 다크
    → "문의가 없을까요?" → --primary 강조 색상
  - 서브카피 (2줄 이내):
      "처음부터 전환율 중심으로 설계하면,
       사이트가 24시간 영업사원이 됩니다."
    → 폰트: 1.05rem / 회색
  - CTA 버튼:
      "어떤 사이트가 필요한지 무료 상담받기 →"
      → primary 배경, 흰색 텍스트, 패딩 14px 28px

[우측]
  - SVG/일러스트: 화면에 방문자가 '문의하기' 버튼 누르는 모습
    또는 모바일/데스크탑 이중 프레임 목업
  - 크기: 420×340px (모바일: 숨김)
```

---

### Section 2 — 공감 체크리스트

```
레이아웃: 2열 카드 그리드 (모바일 1열)
배경: #F8F9FA

[H2] "혹시 이런 상황이세요?" (중앙 정렬)

[카드 × 6]
  각 카드: 흰 배경 / 라운드 8px / 체크박스 아이콘 + 1줄 텍스트
  1. "처음으로 홈페이지를 만들려는데 어디서 시작할지 모른다"
  2. "저렴한 곳에 맡겼다가 완성 후 연락이 끊겼다"
  3. "쇼핑몰인지, 브랜드 사이트인지 모르겠다"
  4. "견적받으러 갔더니 용어가 어려워 뭘 사는지 몰랐다"
  5. "완성 후에도 글 수정하려면 개발자에게 연락해야 한다"
  6. "경쟁사에 비해 온라인 존재감이 너무 약하다"

[결론 문구] border-left --primary 라인
  "하나라도 해당된다면, 사이트 구조부터 다시 생각해야 합니다."
```

---

### Section 3 — 수치 근거

```
레이아웃: 4개 빅 넘버 카드 (수평, 모바일 2×2)
배경: 다크

[카드 × 4] — 흰 텍스트
  1. "0.05초"  / 첫 인상 형성 시간 (--primary 강조)
  2. "0.3%"   / CTA 없는 사이트 전환율
  3. "2.1배"  / 전환율 설계 후 문의 증가 (자사 기준)
  4. "63%"    / 국내 모바일 인터넷 이용 비율
```

---

### Section 4 — 솔루션 (2×2 카드)

```
레이아웃: 2×2 그리드 카드 (모바일 1열)
배경: 흰색

[섹션 헤더]
  H2: "구축 후 이렇게 달라집니다" (좌측 정렬)

[카드 × 4]
  ┌────────────────────────────┐
  │  🎯 아이콘 (2rem)           │
  │  제목 (bold)               │
  │  설명 1줄 (회색)            │
  └────────────────────────────┘

  1. 🎯 전환율 설계  → "방문자가 문의 버튼을 자연스럽게 누르게 됩니다."
  2. 📱 모바일 퍼스트 → "스마트폰 완벽, PC도 깔끔."
  3. ✏️ 직접 편집    → "관리자 페이지로 글·이미지 직접 수정."
  4. 🔍 SEO 기반 설계 → "처음부터 검색 상위 노출 구조로 설계."
```

---

### Section 5 — 구축 유형 카드

```
레이아웃: 4열 유형 카드 (모바일 2×2)
배경: #F8F9FA

[섹션 헤더]
  H2: "어떤 사이트가 필요하신가요?" (중앙)

[유형 카드 × 4]
  ┌──────────────────┐
  │  🏢 아이콘        │
  │  유형명 (bold)    │
  │  "적합한 경우"    │
  │  특징 1줄 (회색)  │
  └──────────────────┘

  1. 🏢 브랜드 사이트   — 회사 소개·포트폴리오 / 신뢰감+SEO
  2. 🛒 쇼핑몰·커머스  — 제품 판매 / 결제·재고 포함
  3. 🔧 서비스 플랫폼   — 예약·멤버십·커뮤니티 / 사용자 계정
  4. 📣 랜딩페이지      — 광고·프로모션 / 단일 CTA, 빠른 론칭

  카드 hover: border --primary, 아이콘 살짝 확대
```

### Section 6 — 진행 프로세스

```
레이아웃: 수직 스텝퍼
배경: 흰색

[스텝 × 4]
  STEP 01  요구사항 정의   (1~2일)
  STEP 02  기획·디자인     (1~2주)
  STEP 03  개발            (2~4주)
  STEP 04  배포·인수인계   (3일)

[하단 메모] "전체 기간: 평균 5~8주" (회색)
```

### Section 7 — 가격표

```
레이아웃: 3열 카드 (모바일 1열 스택)
배경: #F8F9FA

[섹션 도입]
  "기본형 200만원대부터 시작합니다." (H3)
  "정확한 견적은 무료 상담 후 제공됩니다." (회색)

[카드 × 3]
  카드 1 — 랜딩페이지   100만원대~ / 단일 페이지, CTA 중심
  카드 2 — 브랜드 사이트 200만원대~ / 다중 페이지, SEO, 관리자 ★추천
  카드 3 — 커머스·플랫폼 500만원대~ / 결제·회원 시스템

  중간 카드: primary 테두리 + "추천" 배지

[CTA 2] "내 상황에 맞는 견적 받기 →" (중앙, primary 버튼)
```
