# 레이아웃 명세 — `/app-dev` (앱·시스템 개발)

> VIS-196 산출물 | AI 디자이너 작성 | 2026-04-17

---

## 0. 디자인 방향

- **분위기**: 좌절감(엑셀 한계·카오스) → 통제감(체계적 시스템·스프린트 개발)
- **색상**: `--primary` 중심 + 기술적 느낌(청색 계열)
- **특징**: 비포/애프터 2열 비교 카드가 핵심 — 상황→결과 시각화
- **가격**: 4단계 유형별 (랜딩~풀스택) — 범위 넓어서 명확히 분리

## 2. 섹션별 상세 명세

### Section 1 — Hero

```
레이아웃: 2열
배경: 흰색 + 우측 하단 primary 연한 그라디언트

[좌측]
  - 배지: "앱·시스템 개발" (primary pill)
  - H1 (2줄):
      "엑셀로 더 이상
       버틸 수 없는 순간이 있습니다"
    → 폰트: 2.8rem / bold / 다크
    → "버틸 수 없는 순간" → --destructive 강조
  - 서브카피 (3줄, 짧게):
      "직원 10명이 같은 파일을 열었다가 덮어쓰고,
       '앱으로 만들면 좋겠다'고 생각한 지 1년이 넘었다면—
       지금이 바로 그 순간입니다."
    → 1.05rem / 회색
  - CTA 버튼:
      "어떤 시스템이 필요한지 무료 상담받기 →"
      → primary 배경

[우측]
  - SVG/일러스트: 혼란스러운 엑셀 파일 더미 → 깔끔한 앱 화면 전환
    또는: 다중 기기(모바일+태블릿+PC) 목업
  - 크기: 420×360px (모바일: 숨김)
```

### Section 2 — 공감 체크리스트

```
레이아웃: 2열 카드 그리드
배경: #F8F9FA

[H2] "혹시 이런 상황이세요?" (중앙)

[카드 × 6]
  1. "재고·주문·고객 데이터를 아직도 엑셀로 관리한다"
  2. "직원이 늘었는데 업무 공유가 카카오톡과 엑셀뿐이다"
  3. "모바일 앱을 만들고 싶은데 어디에 얼마를 써야 하는지 모른다"
  4. "기존 시스템이 너무 오래돼서 바꾸기 무섭다"
  5. "외주 개발 후 연락이 끊겨서 고생한 적 있다"
  6. "시중 솔루션에는 없는 기능이라 직접 만들어야 할 것 같다"

[결론] border-left --primary
  "2개 이상 해당된다면, 맞춤 개발이 장기적으로 더 저렴합니다."
```

### Section 3 — 수치 근거

```
레이아웃: 4개 빅 넘버 카드 (수평)
배경: 다크 (#1A1A2E)

[카드 × 4] — 흰색, 수치 primary
  1. "43%"  / 맞춤 시스템 도입 후 업무 처리 시간 단축
  2. "72%"  / 수작업 오류 발생률 감소
  3. "38%"  / 모바일 앱 도입 후 고객 재방문율 증가
  4. "92%"  / 스프린트 개발 일정 준수율 (자사 기준)
```

### Section 4 — 개발 영역 비포/애프터

```
레이아웃: 4행 비교 카드 (2열: 상황 → 결과)
배경: 흰색

[섹션 헤더]
  H2: "이렇게 달라집니다" (좌측)

[비교 행 × 4]
  ┌──────────────────────────────────────────────────────────┐
  │  [솔루션 태그]     │  지금 상황 (빨간 배경)  →  개발 후 (primary 배경)  │
  └──────────────────────────────────────────────────────────┘

  행 1: 모바일 앱    — "앱이 없어 고객 불편" → "iOS·Android 동시, 재방문율 상승"
  행 2: 사내 시스템  — "엑셀 10개, 담당자만 앎" → "전 직원 실시간 공유, 오류 70% 감소"
  행 3: API·백엔드   — "외부 서비스 수작업 연동" → "자동 데이터 연동, 사람 손 불필요"
  행 4: 레거시 전환  — "수정도 못 하는 낡은 시스템" → "단계적 이전, 기존 데이터 100% 보전"

  → 수치(70%, 100% 등)는 bold + primary 색상
```

[하단 메모] "전체 기간: 평균 6~16주 / 매주 진행 보고" (회색)

### Section 6 — 가격표

```
레이아웃: 4열 카드 (모바일 2×2)
배경: 흰색

[섹션 도입]
  "기본형 300만원대부터 시작합니다."
  "'이 정도 기능이면 얼마?' — 상담에서 솔직하게 말씀드립니다."

[카드 × 4]
  카드 1 — 랜딩·웹뷰 앱    200만원대~ / 웹사이트+앱 래핑
  카드 2 — 사내 관리 시스템 300만원대~ / 맞춤 기능, 관리자
  카드 3 — 모바일 앱        500만원대~ / iOS·Android+백엔드 ★추천
  카드 4 — 풀스택 플랫폼    1,000만원대~ / 회원·결제·대시보드

  강조: 카드 3에 primary 테두리 + "추천" 배지

[CTA 2] "요구사항 공유하고 견적받기 →" (primary 버튼)
```
