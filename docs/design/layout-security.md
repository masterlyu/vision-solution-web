## 2. 추가 섹션 상세 명세

### [추가 섹션 A] 공감 체크리스트

```
삽입 위치: Hero 섹션 바로 다음 (통계 섹션 앞)
레이아웃: 2열 카드 그리드 (모바일 1열)
배경: #F8F9FA

[H2] "혹시 이런 상황이세요?" (중앙)

[카드 × 7] — 체크박스 아이콘 + 1줄 텍스트
  아이콘 스타일: 미체크 상태, --destructive 테두리 (경고감)
  1. 🔒 "주소창에 자물쇠 표시가 없다"
  2. ⏰ "마지막 보안 점검이 언제인지 모른다"
  3. 📞 "사이트 만든 업체와 연락이 끊겼다"
  4. ❓ "고객이 '이 사이트 안전한가요?'라고 물어본 적 있다"
  5. 🔑 "비밀번호를 한 번도 바꾼 적 없는 관리자 계정이 있다"
  6. 🔌 "워드프레스·플러그인 업데이트를 오래 안 했다"
  7. 🚨 "어느 날 갑자기 사이트가 이상한 페이지로 바뀐 적 있다"

카드 구조:
  ┌─────────────────────────────────┐
  │  [이모지 2rem]  [체크박스 미체크] │
  │  체크 항목 텍스트 (1줄, bold)    │
  └─────────────────────────────────┘
  배경: 흰색 / shadow-sm / radius 8px / padding 16px 20px

[결론 + CTA] border-left --destructive 라인
  "하나라도 해당된다면, 지금 당장 무료 진단을 받으세요."
  [버튼] "지금 무료 보안 진단 받기 →" (URL 입력폼으로 스크롤, --destructive 배경)
```

### [추가 섹션 B] FAQ

```
삽입 위치: 진행 프로세스 섹션 다음 (최종 CTA 앞)
레이아웃: 아코디언 단일 열
배경: #F8F9FA

[H2] "보안 진단 FAQ" (중앙)

[아코디언 × 8]
  각 항목: 질문(bold) + 우측 chevron ▼ + 답변(펼치면 노출)
  항목 간 border-bottom 구분

  Q1. 진단이 진짜 무료인가요? 나중에 비용이 청구되지 않나요?
  Q2. URL만 입력하면 되나요? 관리자 계정이 필요한가요?
  Q3. 진단 중에 사이트가 느려지거나 오류가 생기지 않나요?
  Q4. 리포트에는 어떤 내용이 담기나요?
  Q5. 리포트를 받고 나서 어떻게 해야 하나요?
  Q6. WordPress가 아닌 사이트도 진단되나요?
  Q7. 이미 보안이 잘 된 사이트도 진단하면 의미가 있나요?
  Q8. 진단 후 영업 연락이 오나요?

아코디언 스타일:
  - 닫힌 상태: 질문 + 우측 chevron (↓)
  - 열린 상태: 질문 강조(bold) + chevron (↑) + 답변 텍스트 (회색, padding 16px)
  - 첫 번째 항목 기본 열린 상태
```

### [추가 섹션 C] 가격 안내

```
삽입 위치: FAQ 다음 (최종 CTA 앞)
레이아웃: 3열 카드 + 상단 도입 텍스트
배경: 흰색

[섹션 헤더]
  H2: "취약점 발견 시 개선 비용은?" (좌측)
  서브: "진단은 무료. 개선 작업은 범위에 따라 별도 견적입니다." (회색)

[카드 × 3]
  ┌──────────────────────────────┐
  │  작업 유형 (bold)             │
  │  가격 범위 (2rem, --primary)  │
  │  ─────────────────────       │
  │  • 포함 항목 1               │
  │  • 포함 항목 2               │
  │  • 포함 항목 3               │
  └──────────────────────────────┘

  카드 1 — 기본 보안 설정    30만원~
    • SSL/HTTPS 적용
    • 보안 헤더 6가지 설정
    • 관리자 URL 변경

  카드 2 — 표준 취약점 개선  80만원~   ★추천 배지
    • 기본 설정 전체 포함
    • 플러그인 업데이트
    • 권한·계정 설정 강화

  카드 3 — 심층 보안 강화    200만원~
    • 전체 코드 취약점 스캔
    • WAF 설치
    • 24시간 모니터링 설정

[하단 메모] "진단 리포트 기준으로 정확한 견적을 제공합니다." (회색, 중앙)
```

## 3. 기존 섹션 UI 개선 사항 (브리프 V2 기준)

```
Hero:
  - badge 4개 → 삭제, shield Lottie 아래 CTA 버튼 1개로 대체
  - H1: span 통합 (2줄 이내)
  - 서브카피: 1줄로 압축 ("국내 중소기업 73%가 지금 해킹에 노출되어 있습니다.")

Stats 섹션:
  - 각 stat 설명 2줄 → 1줄 압축

Danger Cases 카드:
  - `desc` 필드 삭제, `case` 배지만 유지

Checklist 카드:
  - simple/warn 설명 삭제
  - 이모지(4rem) + 질문 제목 1줄 + 위험도 바 구조

Process 스텝퍼:
  - desc 삭제, 5단어 이내 동사형 제목만
  - overline 삭제

Site Types 카드:
  - note 삭제, 이모지+라벨+O/△/X 아이콘만

CTA Form:
  - overline 삭제
  - 혜택 4개: 이모지+title만, desc 삭제
```
